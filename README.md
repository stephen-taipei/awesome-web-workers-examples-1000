**Awesome Web Workers Examples 1000** 是一個收集 Web Workers 技術實作範例的專案，目標建立 1000 個實用的範例，涵蓋從基礎計算到進階多執行緒應用的完整教學資源，幫助開發者深入理解 Web Workers 技術。

### 專案特色

- **實用導向**：每個範例都是可獨立運行的完整應用
- **漸進學習**：從簡單到複雜的系統化學習路徑
- **效能優化**：展示多執行緒的效能優勢與最佳實踐
- **現代技術**：採用最新的 Web Workers API 與相關技術
- **完整文檔**：每個範例都有詳細說明與使用指南

## Web Workers 介紹

### 什麼是 Web Workers？

Web Workers 是瀏覽器提供的多執行緒機制，讓 JavaScript 可以在背景執行緒中運行，不會阻塞主執行緒 (UI 執行緒)，從而實現真正的並行處理。

### Worker 類型

| 類型 | 說明 | 使用場景 |
|------|------|----------|
| **Dedicated Worker** | 專屬單一頁面 | 大量數據處理 |
| **Shared Worker** | 多頁面共享 | 跨分頁通訊 |
| **Service Worker** | 離線與快取 | PWA 應用 |

### 基本 API

```javascript
// 建立 Worker
const worker = new Worker('worker.js');

// 傳送訊息
worker.postMessage({ type: 'start', data: payload });

// 接收訊息
worker.onmessage = (event) => {
  console.log('結果:', event.data);
};

// 終止 Worker
worker.terminate();
```
