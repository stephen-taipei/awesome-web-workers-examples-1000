About Numerical Differentiation

Numerical differentiation approximates derivatives from discrete data points using finite difference methods.

Finite Difference Formulas

Method Formula Accuracy
Forward [f(x+h) - f(x)] / h O(h)
Backward [f(x) - f(x-h)] / h O(h)
Central [f(x+h) - f(x-h)] / 2h O(h²)
Five-Point [-f(x+2h) + 8f(x+h) - 8f(x-h) + f(x-2h)] / 12h O(h⁴)

Second Derivative

Formula: f''(x) ≈ [f(x+h) - 2f(x) + f(x-h)] / h²
