About Map-Reduce

Map-Reduce is a programming model for processing large datasets in parallel. Data is first transformed by map functions running on multiple workers, then combined by reduce functions to produce the final result.

Map-Reduce Pattern:

Map-Reduce Flow:
  Input Data → Split → Map → Shuffle → Reduce → Output

Map Phase:
  map(key, value) → [(key', value'), ...]
  - Processes each input element independently
  - Emits intermediate key-value pairs
  - Runs in parallel across workers

Shuffle Phase:
  - Groups intermediate values by key
  - Sorts and partitions data for reducers

Reduce Phase:
  reduce(key, [values]) → result
  - Combines all values for each key
  - Produces final aggregated results

Example - Word Count:
  map("hello world") → [("hello", 1), ("world", 1)]
  reduce("hello", [1, 1, 1]) → ("hello", 3)
