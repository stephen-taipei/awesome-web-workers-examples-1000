About SVD

Singular Value Decomposition factorizes any m×n matrix A into A = UΣVT

  • U: m×m orthogonal matrix (left singular vectors)
  • Σ: m×n diagonal matrix (singular values σ₁ ≥ σ₂ ≥ ... ≥ 0)
  • V: n×n orthogonal matrix (right singular vectors)

Algorithms

AlgorithmComplexityBest For
Power IterationO(mn·k·iter)Top k singular values
Golub-KahanO(mn²)Full decomposition
One-Sided JacobiO(mn²)High accuracy

Applications

  • Principal Component Analysis (PCA)
  • Image compression
  • Recommendation systems
  • Solving linear least squares
  • Computing matrix rank and pseudoinverse
