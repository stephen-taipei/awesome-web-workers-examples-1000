About Matrix Determinant

The determinant det(A) is a scalar value that encodes properties of a square matrix.

Algorithms

AlgorithmComplexityBest For
LU DecompositionO(n³)General use, numerical
CofactorO(n!)Small matrices (n≤10)
BareissO(n³)Integer matrices

Properties

  • det(AB) = det(A) × det(B)
  • det(AT) = det(A)
  • det(A-1) = 1/det(A)
  • det(cA) = cndet(A) for n×n matrix
  • det(A) = 0 ⟺ A is singular
