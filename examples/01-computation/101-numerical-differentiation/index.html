About Numerical Differentiation

Numerical differentiation approximates the derivative of a function using finite differences.

Difference Formulas

Forward Difference
f'(x) ≈ [f(x+h) - f(x)] / h
O(h)
Backward Difference
f'(x) ≈ [f(x) - f(x-h)] / h
O(h)
Central Difference
f'(x) ≈ [f(x+h) - f(x-h)] / 2h
O(h²)
Five-Point Formula
f'(x) ≈ [-f(x+2h) + 8f(x+h) - 8f(x-h) + f(x-2h)] / 12h
O(h⁴)

Error Analysis

  • Truncation error: From Taylor series approximation
  • Round-off error: From finite precision arithmetic
  • Optimal h: Balances both error sources (typically h ≈ √ε for central)
