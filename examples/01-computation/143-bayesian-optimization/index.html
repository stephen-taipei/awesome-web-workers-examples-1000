About Bayesian Optimization

Bayesian Optimization (BO) is a strategy for global optimization of black-box functions. It builds a probabilistic model (Gaussian Process) of the objective function and uses an Acquisition Function to decide where to sample next.

Core Components:

1. Gaussian Process (GP):
   Models f(x) ~ GP(m(x), k(x, x'))
   Provides mean μ(x) and variance σ²(x)

2. Acquisition Function (Expected Improvement):
   EI(x) = (μ(x) - f_best - ξ)Φ(Z) + σ(x)φ(Z)
   where Z = (μ(x) - f_best - ξ) / σ(x)

3. Strategy:
   Select x_next = argmax EI(x)
