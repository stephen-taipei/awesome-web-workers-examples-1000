#071 Cholesky Decomposition

Cholesky, LDL, and modified Cholesky for symmetric positive definite matrices

About Cholesky Decomposition

For symmetric positive definite matrix A, factorize A = LLT where L is lower triangular with positive diagonal entries.

Algorithms

AlgorithmFormFeatures
Standard CholeskyA = LLTRow-by-row computation
LDLTA = LDLTAvoids square roots
Cholesky-BanachiewiczA = LLTRow-oriented algorithm
Cholesky-CroutA = LLTColumn-oriented algorithm

Applications

  • Solving symmetric positive definite linear systems
  • Monte Carlo simulation (generating correlated random variables)
  • Kalman filters
  • Computing determinants: det(A) = det(L)2
  • Matrix inversion for covariance matrices

Requirements

Matrix must be symmetric (A = AT) and positive definite (all eigenvalues > 0).
