About These Numbers

Euler Numbers |E(n)|

Appear in Taylor series of sec(x). Odd indices are 0.

1, 1, 5, 61, 1385, 50521, ...

Zigzag Numbers A(n)

Count alternating permutations of n elements.

1, 1, 1, 2, 5, 16, 61, 272, ...

Tangent Numbers T(n)

Coefficients in Taylor series of tan(x).

1, 2, 16, 272, 7936, ...

Mathematical Relationships

Euler numbers E(n) satisfy:

  • E(2n+1) = 0 for all n
  • |E(2n)| = A(2n) (zigzag number at even index)
  • sec(x) = Σ E(n) * xn / n!

Zigzag numbers A(n) count:

  • Alternating permutations: a1 < a2 > a3 < a4 > ...
  • A(0) = A(1) = 1
  • A(n) = sec(x) + tan(x) Taylor coefficients

Tangent numbers T(n):

  • T(n) = A(2n+1) = zigzag number at odd index
  • tan(x) = Σ T(n) * x2n+1 / (2n+1)!
