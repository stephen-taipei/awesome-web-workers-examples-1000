About Grid Search

Grid Search is an exhaustive search method that evaluates a function at every point within a predefined grid of parameters. It is commonly used for hyperparameter optimization to find the best combination of parameters.

Method:

Minimize f(x, y) over:
x ∈ [x_min, x_max] with step Δx
y ∈ [y_min, y_max] with step Δy

Total evaluations = N_x × N_y
Complexity: O(N²) for 2D
