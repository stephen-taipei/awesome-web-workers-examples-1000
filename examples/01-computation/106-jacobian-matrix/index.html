About the Jacobian Matrix

The Jacobian matrix J is the matrix of all first-order partial derivatives of a vector-valued function F: ℝⁿ → ℝᵐ.

Definition

J(F) = [∂fᵢ/∂xⱼ]
where Jᵢⱼ = ∂fᵢ/∂xⱼ (m×n matrix)

Numerical Approximation

Central Difference
∂fᵢ/∂xⱼ ≈ [fᵢ(x + heⱼ) - fᵢ(x - heⱼ)] / 2h
O(h²)
Forward Difference
∂fᵢ/∂xⱼ ≈ [fᵢ(x + heⱼ) - fᵢ(x)] / h
O(h)

Properties & Applications

  • Linearization: F(x + Δx) ≈ F(x) + J·Δx near x
  • Newton's Method: Solve F(x) = 0 via x_{n+1} = x_n - J⁻¹F(x_n)
  • Change of Variables: Volume element scales by |det(J)|
  • Sensitivity Analysis: Jᵢⱼ measures sensitivity of fᵢ to xⱼ
