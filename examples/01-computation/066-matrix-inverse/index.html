About Matrix Inverse

Finding A-1 such that A × A-1 = I

Algorithms

AlgorithmComplexityBest For
Gauss-JordanO(n³)General use
LU DecompositionO(n³)Multiple systems
AdjugateO(n!)Small matrices (n≤5)

Properties

  • (A-1)-1 = A
  • (AB)-1 = B-1A-1
  • (AT)-1 = (A-1)T
  • det(A-1) = 1/det(A)
