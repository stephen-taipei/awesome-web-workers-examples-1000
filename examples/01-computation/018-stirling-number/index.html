About Stirling Numbers

First Kind |s(n,k)|

Counts permutations of n elements with exactly k cycles (unsigned).

Recurrence: s(n,k) = s(n-1,k-1) + (n-1)·s(n-1,k)

Second Kind S(n,k)

Counts partitions of n elements into exactly k non-empty subsets.

Recurrence: S(n,k) = S(n-1,k-1) + k·S(n-1,k)
