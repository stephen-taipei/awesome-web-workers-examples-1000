About Linear Regression

Simple Linear Regression fits a straight line to data points to model the relationship between a dependent variable Y and an independent variable X.

The Model

y = β₀ + β₁x + ε

Where β₀ is the intercept, β₁ is the slope, and ε is the error term.

OLS Formulas

Parameter Formula
Slope (β₁) Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
Intercept (β₀) ȳ - β₁x̄
R² 1 - SSR/SST

R² Interpretation

R² Value Interpretation
0.9 - 1.0 Excellent fit
0.7 - 0.9 Good fit
0.5 - 0.7 Moderate fit
0.3 - 0.5 Weak fit
0.0 - 0.3 Poor fit
