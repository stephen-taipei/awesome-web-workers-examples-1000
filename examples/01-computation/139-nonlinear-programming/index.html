About Nonlinear Programming

Nonlinear Programming (NLP) optimizes nonlinear objective functions subject to nonlinear constraints. Sequential Quadratic Programming (SQP) is a powerful method that solves a sequence of quadratic subproblems to approximate the original NLP.

NLP Standard Form:

Minimize:   f(x)
Subject to: hᵢ(x) = 0,   i = 1,...,p  (equality)
            gⱼ(x) ≤ 0,   j = 1,...,m  (inequality)

SQP Method - At each iteration k:

1. Form QP subproblem:
   min  ∇f(xₖ)ᵀd + ½dᵀBₖd
   s.t. ∇hᵢ(xₖ)ᵀd + hᵢ(xₖ) = 0
        ∇gⱼ(xₖ)ᵀd + gⱼ(xₖ) ≤ 0

2. Solve QP for search direction d

3. Line search: xₖ₊₁ = xₖ + αd

4. Update Hessian approximation Bₖ₊₁
   (BFGS or damped BFGS)

Merit Function (L₁ penalty):
  φ(x; μ) = f(x) + μ Σ|hᵢ(x)| + μ Σmax(0, gⱼ(x))

KKT Conditions:
  ∇f(x*) + Σλᵢ∇hᵢ(x*) + Σμⱼ∇gⱼ(x*) = 0
  hᵢ(x*) = 0,  gⱼ(x*) ≤ 0
  μⱼ ≥ 0,  μⱼgⱼ(x*) = 0
