About Digital Root

The digital root is obtained by repeatedly summing digits until a single digit remains.

Example: 9875 → 9+8+7+5=29 → 2+9=11 → 1+1=2, so dr(9875)=2

Formula: dr(n) = 1 + ((n-1) mod 9) for n > 0
