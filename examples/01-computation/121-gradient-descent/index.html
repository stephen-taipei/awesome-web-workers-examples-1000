About Gradient Descent

Gradient descent is an iterative optimization algorithm that finds local minima of differentiable functions by following the negative gradient direction.

Update Rule

x_{t+1} = x_t - α ∇f(x_t)
α = learning rate, ∇f = gradient of objective function

Learning Rate Schedules

Constant
α_t = α₀
Time Decay
α_t = α₀ / (1 + kt)
Step Decay
α_t = α₀ × 0.5^⌊t/N⌋
Exponential
α_t = α₀ × γᵗ

Convergence Conditions

  • Gradient near zero: ||∇f(x)|| < ε
  • Small step size: ||x_{t+1} - x_t|| < ε
  • Function value stable: |f(x_{t+1}) - f(x_t)| < ε

Challenges

  • Learning rate selection: Too small = slow, too large = diverge
  • Local minima: May converge to non-global minimum
  • Saddle points: Slow progress in flat regions
  • Ill-conditioning: Different scales require different rates
