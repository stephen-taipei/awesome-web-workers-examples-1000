About Condition Number

The condition number κ(A) measures how sensitive the solution of Ax = b is to perturbations in A or b. A large condition number indicates an ill-conditioned matrix.

Interpretation

  • κ(A) ≈ 1: Well-conditioned (orthogonal matrices)
  • κ(A) ~ 10^k: May lose ~k digits of precision
  • κ(A) ~ 10^16: Effectively singular (double precision)
  • κ(I) = 1: Identity is perfectly conditioned

Applications

  • Numerical stability analysis
  • Error bound estimation for Ax = b
  • Preconditioning strategy selection
  • Matrix regularization decisions
