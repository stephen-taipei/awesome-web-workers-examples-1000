About Matrix Square Root

The matrix square root √A (or A^(1/2)) is a matrix B such that B² = A. For positive semidefinite matrices, the principal square root exists and is unique.

Properties

  • (√A)² = A: Defining property
  • √I = I: Identity is its own root
  • √(diag(a,b)) = diag(√a, √b): Diagonal case
  • det(√A) = √det(A): Determinant relation

Applications

  • Whitening/decorrelation in signal processing
  • Matrix geometric mean: A#B = A^(1/2)(A^(-1/2)BA^(-1/2))^(1/2)A^(1/2)
  • Covariance matrix operations
  • Matrix interpolation and morphing
