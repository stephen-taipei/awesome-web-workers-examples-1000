About ARIMA

ARIMA (AutoRegressive Integrated Moving Average) is a popular time series forecasting method that combines autoregression, differencing, and moving averages.

Model Components

Component Parameter Description
AR p Number of autoregressive terms
I d Degree of differencing
MA q Number of moving average terms

Common Models

Model Use Case
ARIMA(1,0,0) Simple AR process
ARIMA(0,1,1) Simple exponential smoothing
ARIMA(1,1,1) Mixed with differencing
