About B-Spline Curves

B-splines (Basis splines) are piecewise polynomial curves defined by control points and a knot vector, widely used in computer graphics and CAD.

B-Spline Formula

C(t) = Σᵢ Pᵢ · Nᵢ,ₖ(t)
where Pᵢ are control points and Nᵢ,ₖ(t) are basis functions

Cox-de Boor Recursion

Base Case (k=0)
Nᵢ,₀(t) = 1 if tᵢ ≤ t < tᵢ₊₁, else 0
Recursive (k≥1)
Nᵢ,ₖ(t) = αᵢ,ₖNᵢ,ₖ₋₁ + (1-αᵢ₊₁,ₖ)Nᵢ₊₁,ₖ₋₁
Blend Coefficient
αᵢ,ₖ = (t - tᵢ) / (tᵢ₊ₖ - tᵢ)

Knot Vector Types

  • Uniform: Evenly spaced knots [0, 1, 2, ..., n+k]
  • Clamped (Open): First and last k+1 knots repeated for endpoint interpolation
  • Custom: User-defined knot sequence

Properties

  • Local control: Moving one control point affects only nearby curve segments
  • Partition of unity: Basis functions sum to 1
  • Non-negativity: Nᵢ,ₖ(t) ≥ 0
  • C^(k-1) continuity for degree k with simple knots
