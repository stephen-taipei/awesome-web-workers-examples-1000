About Hadamard Product

The Hadamard (or Schur) product is the element-wise multiplication of two matrices of the same dimensions.

Operations

OperationSymbolFormula
Hadamard ProductA ∘ BC[i,j] = A[i,j] × B[i,j]
Hadamard DivisionA ⊘ BC[i,j] = A[i,j] / B[i,j]
Hadamard PowerA.^nC[i,j] = A[i,j]^n

Properties

  • Commutative: A ∘ B = B ∘ A
  • Associative: (A ∘ B) ∘ C = A ∘ (B ∘ C)
  • Distributive: A ∘ (B + C) = A ∘ B + A ∘ C
  • Identity: A ∘ J = A (J = all ones)

Applications

  • Image processing (masking, blending)
  • Neural networks (attention mechanisms)
  • Feature weighting in ML
  • Signal processing
