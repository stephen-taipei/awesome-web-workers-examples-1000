About Partial Derivatives

Partial derivatives measure how a multivariable function changes with respect to one variable while holding others constant.

First-Order Partial Derivatives

∂f/∂x (Central)
[f(x+h, y) - f(x-h, y)] / 2h
O(h²)
∂f/∂y (Central)
[f(x, y+h) - f(x, y-h)] / 2h
O(h²)

Second-Order Partial Derivatives

∂²f/∂x²
[f(x+h, y) - 2f(x, y) + f(x-h, y)] / h²
O(h²)
∂²f/∂y²
[f(x, y+h) - 2f(x, y) + f(x, y-h)] / h²
O(h²)
∂²f/∂x∂y (Mixed)
[f(x+h, y+h) - f(x+h, y-h) - f(x-h, y+h) + f(x-h, y-h)] / 4h²
O(h²)

Applications

  • Gradient: ∇f = (∂f/∂x, ∂f/∂y) points in steepest ascent direction
  • Optimization: Critical points where ∇f = 0
  • Physics: Heat flow, wave propagation, fluid dynamics
  • Machine Learning: Backpropagation uses partial derivatives
