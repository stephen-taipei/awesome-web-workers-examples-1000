About PDEs

Partial Differential Equations describe phenomena involving multiple independent variables like space and time.

Equation Types

TypeEquationApplication
Heat∂u/∂t = α∇²uDiffusion, conduction
Wave∂²u/∂t² = c²∇²uVibrations, acoustics
Laplace∇²u = 0Steady-state, potential
Poisson∇²u = fElectrostatics, gravity
Advection∂u/∂t + v·∇u = 0Transport, flow

Stability Conditions

Heat: r = α×dt/dx² ≤ 0.5

Wave: C = c×dt/dx ≤ 1 (CFL condition)
