About Linear Programming

Linear Programming (LP) optimizes a linear objective function subject to linear equality and inequality constraints. The Simplex method efficiently finds optimal solutions by traversing vertices of the feasible region.

Standard Form:

Maximize:   z = c₁x₁ + c₂x₂ + ... + cₙxₙ

Subject to:
  a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
  a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ ≤ b₂
  ...
  aₘ₁x₁ + aₘ₂x₂ + ... + aₘₙxₙ ≤ bₘ

  x₁, x₂, ..., xₙ ≥ 0

Simplex Method:
1. Convert to standard form (add slack variables)
2. Find initial basic feasible solution
3. While not optimal:
   a. Select entering variable (most positive cⱼ - zⱼ)
   b. Select leaving variable (minimum ratio test)
   c. Pivot to new basic solution
4. Read optimal solution from tableau
