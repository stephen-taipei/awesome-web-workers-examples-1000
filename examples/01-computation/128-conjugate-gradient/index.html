About Conjugate Gradient

The Conjugate Gradient method generates conjugate search directions that avoid redundant exploration. For quadratic functions, it converges in at most n iterations. For general functions, nonlinear CG variants like Fletcher-Reeves and Polak-Ribière are used.

Nonlinear CG Update:

d_0 = -∇f(x_0)                    (Initial direction = steepest descent)
x_{k+1} = x_k + α_k d_k           (Line search)
d_{k+1} = -∇f(x_{k+1}) + β_k d_k  (Conjugate direction)

β formulas:
  Fletcher-Reeves: β = ||∇f_{k+1}||² / ||∇f_k||²
  Polak-Ribière:   β = ∇f_{k+1}^T (∇f_{k+1} - ∇f_k) / ||∇f_k||²
  Hestenes-Stiefel: β = ∇f_{k+1}^T (∇f_{k+1} - ∇f_k) / d_k^T (∇f_{k+1} - ∇f_k)
