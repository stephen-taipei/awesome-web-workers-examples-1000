About Banded Matrices

A banded matrix has non-zero elements only near the main diagonal. Storage and operations can be optimized.

Bandwidth Parameters

  • Lower bandwidth (p): Number of non-zero diagonals below main
  • Upper bandwidth (q): Number of non-zero diagonals above main
  • Total bandwidth: p + q + 1
