About Matrix Exponential

The matrix exponential e^A is defined as the infinite series: e^A = I + A + A²/2! + A³/3! + ...

Properties

  • e^0 = I: Identity matrix
  • det(e^A) = e^tr(A): Determinant formula
  • e^(A+B) = e^A e^B: Only if AB = BA
  • (e^A)⁻¹ = e^(-A): Inverse

Applications

  • Solving linear ODEs: dx/dt = Ax → x(t) = e^(At)x(0)
  • Quantum mechanics (unitary evolution)
  • Control theory
  • Lie groups and Lie algebras
