About Tridiagonal Systems

A tridiagonal matrix has non-zero elements only on the main diagonal and the two adjacent diagonals.

Thomas Algorithm (TDMA)

  • Complexity: O(n) time, O(n) space
  • Forward sweep: Eliminate sub-diagonal
  • Back substitution: Solve from last to first
  • Stability: Stable for diagonally dominant matrices
