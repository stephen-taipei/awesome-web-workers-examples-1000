About Polynomial Regression

Polynomial Regression fits a polynomial curve to data points, allowing for non-linear relationships while still using linear algebra techniques.

The Model

y = β₀ + β₁x + β₂x² + ... + βₙxⁿ

Degree Selection

Degree Curve Type Parameters
1 Linear (straight line) 2
2 Quadratic (parabola) 3
3 Cubic 4
n n-th degree polynomial n+1

Model Selection Criteria

  • AIC (Akaike): Balances fit vs complexity
  • BIC (Bayesian): Stronger penalty for parameters
  • Adjusted R²: R² adjusted for parameters
Warning: High-degree polynomials may overfit the data. Use cross-validation or information criteria to select the optimal degree.
