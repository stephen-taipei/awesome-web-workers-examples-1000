About Matrix Transpose

Computing AT where AT[i][j] = A[j][i]

Algorithms

AlgorithmSpaceBest For
StandardO(mn)Any matrix
In-placeO(1)Square matrices
BlockO(B²)Cache efficiency
RecursiveO(log n)Cache-oblivious

Properties

  • (AT)T = A
  • (A + B)T = AT + BT
  • (AB)T = BTAT
  • det(AT) = det(A)
