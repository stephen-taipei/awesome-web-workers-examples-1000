About Newton's Method

Newton's method (Newton-Raphson) finds roots of f(x) = 0 using the tangent line at each iteration to approximate the next guess.

Algorithm

x_{n+1} = x_n - f(x_n) / f'(x_n)

Properties

PropertyValue
ConvergenceQuadratic (when converges)
Order2 (error squares each step)
Requirementf'(x) ≠ 0 near root
Error relation|e_{n+1}| ≈ C|e_n|²

Convergence Behavior

  • Quadratic: Digits of accuracy roughly double each iteration
  • Basin of attraction: Converges if x₀ is "close enough"
  • Can diverge: Bad initial guess or f'(x) ≈ 0
  • Multiple roots: Slower (linear) convergence

Comparison with Bisection

  • Pro: Much faster when it works (quadratic vs linear)
  • Pro: Only needs one starting point
  • Con: Requires derivative (or approximation)
  • Con: Can diverge or cycle
