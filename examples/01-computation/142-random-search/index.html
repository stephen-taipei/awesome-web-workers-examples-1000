About Random Search

Random Search is a family of numerical optimization methods that do not require the gradient of the problem to be optimized. It can hence be used on functions that are not continuous or differentiable.

Pure Random Search:

1. Sample x uniformly from [lower, upper].
2. Evaluate f(x).
3. Update best solution if f(x) < f(best).
4. Repeat N times.

Probability of finding a point in top 1% volume:
P = 1 - (0.99)^N
