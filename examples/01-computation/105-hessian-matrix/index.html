About the Hessian Matrix

The Hessian matrix H is the square matrix of all second-order partial derivatives of a scalar-valued function.

Definition

H(f) = [∂²f/∂xᵢ∂xⱼ]
where Hᵢⱼ = ∂²f/∂xᵢ∂xⱼ

Numerical Approximation

Diagonal Elements
∂²f/∂xᵢ² ≈ [f(x+heᵢ) - 2f(x) + f(x-heᵢ)] / h²
Off-Diagonal Elements
∂²f/∂xᵢ∂xⱼ ≈ [f(x+heᵢ+heⱼ) - f(x+heᵢ-heⱼ) - f(x-heᵢ+heⱼ) + f(x-heᵢ-heⱼ)] / 4h²

Properties & Applications

  • Symmetry: H is symmetric if f has continuous second derivatives (Schwarz's theorem)
  • Convexity: f is convex iff H ≽ 0 (positive semidefinite) everywhere
  • Critical Points: At ∇f = 0, eigenvalues of H determine nature of critical point
  • Newton's Method: Uses H⁻¹∇f for optimization step direction
