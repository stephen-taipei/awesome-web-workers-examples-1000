About Mode

The mode is the value that appears most frequently in a dataset.

Modality Types

  • Unimodal: One mode (single peak)
  • Bimodal: Two modes
  • Multimodal: Multiple modes
  • No mode: All values equally frequent

Mode vs Mean vs Median

Measure Best For Categorical Data
Mode Most common value Yes
Mean Average value No
Median Middle value Ordinal only
