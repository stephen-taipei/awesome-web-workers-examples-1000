About Simulated Annealing

Simulated Annealing is a probabilistic optimization algorithm inspired by the annealing process in metallurgy. It can escape local minima by occasionally accepting worse solutions, with the probability decreasing as the "temperature" cools down.

Algorithm:

1. Initialize: x = random point, T = T_initial
2. For each iteration:
   a. Generate neighbor: x' = x + random perturbation
   b. Calculate ΔE = f(x') - f(x)
   c. If ΔE < 0: accept x' (better solution)
   d. Else: accept with probability P = exp(-ΔE/T)
   e. Cool down: T = cooling_rate × T
3. Repeat until T < T_min or max iterations reached

Cooling Schedules:
  Exponential: T_k = T_0 × α^k           (α ≈ 0.95-0.99)
  Linear:      T_k = T_0 - k × (T_0 - T_min) / k_max
  Logarithmic: T_k = T_0 / (1 + log(1 + k))
