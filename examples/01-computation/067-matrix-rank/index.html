About Matrix Rank

The rank is the dimension of the column space (or row space) of a matrix.

Key Properties

  • rank(A) ≤ min(m, n) for m×n matrix
  • rank(A) = rank(AT)
  • rank(AB) ≤ min(rank(A), rank(B))
  • A is full rank if rank(A) = min(m, n)
