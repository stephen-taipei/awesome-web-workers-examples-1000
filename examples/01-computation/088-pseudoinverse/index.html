About Moore-Penrose Pseudoinverse

The pseudoinverse A⁺ generalizes the concept of matrix inverse to rectangular and singular matrices. It provides the least-squares solution to Ax = b.

Definition

A⁺ is the unique matrix satisfying the Moore-Penrose conditions:

  • AA⁺A = A
  • A⁺AA⁺ = A⁺
  • (AA⁺)ᵀ = AA⁺
  • (A⁺A)ᵀ = A⁺A

SVD Method

If A = UΣVᵀ, then A⁺ = VΣ⁺Uᵀ where Σ⁺ has 1/σᵢ on the diagonal for non-zero singular values.

Special Cases

Matrix TypePseudoinverse
Invertible (n×n)A⁺ = A⁻¹
Full column rank (m > n)A⁺ = (AᵀA)⁻¹Aᵀ
Full row rank (m < n)A⁺ = Aᵀ(AAᵀ)⁻¹
Rank deficientUse SVD method

Applications

  • Least squares solutions: x = A⁺b
  • Minimum norm solutions
  • Linear regression
  • System identification
