About Golden Section Search

Golden Section Search is a technique for finding an extremum (minimum or maximum) of a function inside a specified interval. It works for unimodal functions by successively narrowing the range of values inside which the extremum is known to exist.

Algorithm Steps:

1. Start with interval [a, b] containing the minimum.
2. Golden ratio φ = (1 + √5) / 2 ≈ 1.61803
3. Compute test points:
   x₁ = b - (b - a) / φ
   x₂ = a + (b - a) / φ
4. Compare f(x₁) and f(x₂):
   - If f(x₁) < f(x₂): New interval [a, x₂]
   - If f(x₁) > f(x₂): New interval [x₁, b]
5. Repeat until (b - a) < tolerance.
