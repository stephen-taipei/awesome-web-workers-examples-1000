About Multiple Regression

Multiple Linear Regression models the relationship between a dependent variable and multiple independent variables.

The Model

y = β₀ + β₁x₁ + β₂x₂ + ... + βₚxₚ + ε

Key Diagnostics

Metric Description
VIF Variance Inflation Factor (multicollinearity)
F-statistic Overall model significance
t-statistics Individual coefficient significance

VIF Interpretation

VIF Value Interpretation
1 No multicollinearity
1-5 Moderate, acceptable
5-10 High, concerning
>10 Severe, problematic
