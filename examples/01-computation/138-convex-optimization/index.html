About Convex Optimization

Convex optimization finds the global minimum of a convex function over a convex set. The Interior Point Method (Barrier Method) handles inequality constraints by adding a logarithmic barrier to the objective, then gradually reducing its influence to approach the constrained optimum.

Barrier Method:

Original Problem:
  Minimize:   f(x)
  Subject to: gᵢ(x) ≤ 0,  i = 1,...,m
              Ax = b

Barrier Problem:
  Minimize:   t·f(x) - Σ log(-gᵢ(x))

Where t > 0 is the barrier parameter

Algorithm (Path Following):
1. Initialize with strictly feasible x, t = t₀
2. Centering step: Minimize barrier problem
   using Newton's method
3. Increase t: t ← μ·t  (μ > 1)
4. Repeat until m/t < ε (duality gap bound)

Newton Step for Centering:
  Δx = -H⁻¹∇φ(x)

Where:
  φ(x) = t·f(x) - Σ log(-gᵢ(x))
  H = t·∇²f + Σ (1/gᵢ²)∇gᵢ∇gᵢᵀ - Σ (1/gᵢ)∇²gᵢ

Convergence: O(√m · log(m/ε))
