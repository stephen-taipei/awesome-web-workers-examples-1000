About Matrix Logarithm

The matrix logarithm log(A) is defined such that e^(log(A)) = A. It exists when A has no negative real eigenvalues.

Algorithms

PadéInverse scaling + rational approxGeneral case

Eigendecomplog(A) = V log(Λ) V⁻¹Diagonalizable

Properties

  • e^(log(A)) = A: Inverse of matrix exponential
  • log(I) = 0: Zero matrix
  • tr(log(A)) = log(det(A)): Trace formula
  • log(AB) ≠ log(A) + log(B): In general

Applications

  • Interpolation between rotation matrices
  • Computing matrix roots: A^(1/n) = e^(log(A)/n)
  • Lie algebra computations
  • Geometric mean of matrices
