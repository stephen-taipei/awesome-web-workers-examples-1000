About Newton's Method

Newton's method uses second-order derivative information (Hessian matrix) to achieve faster convergence than first-order methods. It approximates the function locally as a quadratic and finds the optimum in one step for quadratic functions.

Update Rule:

x_{k+1} = x_k - H(x_k)^{-1} * ∇f(x_k)

Where:
  H(x) = Hessian matrix (second derivatives)
  ∇f(x) = Gradient vector (first derivatives)

For 2D: H = | ∂²f/∂x²    ∂²f/∂x∂y |
            | ∂²f/∂y∂x   ∂²f/∂y²  |
