About Sparse Matrices

Sparse matrices have mostly zero elements. Using compressed formats saves memory and speeds up operations.

Storage Formats

FormatDescriptionBest For
CSRCompressed Sparse RowRow slicing, Ax
CSCCompressed Sparse ColumnColumn slicing, A^Tx
COOCoordinate ListConstruction, conversion

Memory Comparison

Dense: n² elements. Sparse (CSR): 2×nnz + n+1 elements, where nnz = non-zero count.

For 1% density: CSR uses ~2% of dense storage!

Applications

  • Scientific computing (FEM, FDM)
  • Graph algorithms (adjacency matrices)
  • Machine learning (feature matrices)
  • Network analysis
