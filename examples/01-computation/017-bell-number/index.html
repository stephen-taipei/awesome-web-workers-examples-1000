About Bell Numbers

Bell numbers count the number of ways to partition a set of n elements. They are calculated using the Bell Triangle (also known as Aitken's array).

Bell Triangle Construction

The Bell Triangle is constructed as follows:

  1. Row 0 starts with 1
  2. The first element of each row is the last element of the previous row
  3. Each subsequent element is the sum of the element to its left and the element above the left element
  4. B(n) = first element of row n
Row 0:  1
Row 1:  1   2
Row 2:  2   3   5
Row 3:  5   7  10  15
Row 4: 15  20  27  37  52

The first element of each row gives B(0)=1, B(1)=1, B(2)=2, B(3)=5, B(4)=15, ...
