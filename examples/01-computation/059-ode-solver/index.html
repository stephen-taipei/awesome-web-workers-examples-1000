About ODE Solvers

Ordinary Differential Equations (ODEs) describe how quantities change over time. Numerical methods approximate solutions when analytical solutions are unavailable.

Solution Methods

Method Formula Accuracy
Euler y_{n+1} = y_n + h·f(t_n, y_n) O(h)
Midpoint y_{n+1} = y_n + h·f(t_n + h/2, y_n + h/2·k₁) O(h²)
Heun y_{n+1} = y_n + h/2·(k₁ + k₂) O(h²)
RK4 y_{n+1} = y_n + h/6·(k₁ + 2k₂ + 2k₃ + k₄) O(h⁴)
RK45 Adaptive step with error control User-specified

Runge-Kutta 4 Coefficients

k₁ = f(t, y)

k₂ = f(t + h/2, y + h/2·k₁)

k₃ = f(t + h/2, y + h/2·k₂)

k₄ = f(t + h, y + h·k₃)
