About Wavelet Transform

Wavelet transforms provide time-frequency analysis of signals, capturing both frequency content and its location in time. Unlike Fourier transforms, wavelets can detect transient features.

Haar Wavelet (Simplest)

Low-pass: h = [1/√2, 1/√2]
High-pass: g = [1/√2, -1/√2]
Averaging (approximation) and differencing (detail)

Decomposition Process

Approximation (cA)
cA[k] = Σ x[n]·h[n-2k]
Detail (cD)
cD[k] = Σ x[n]·g[n-2k]

Applications

  • Signal denoising: Threshold small detail coefficients
  • Data compression: Keep significant coefficients only
  • Edge detection: Large detail coefficients indicate edges
  • Feature extraction: Multi-resolution analysis

Daubechies Wavelets

More complex wavelets with better frequency localization. DB2 uses 4 coefficients, DB4 uses 8 coefficients, providing smoother analysis.
