About Matrix Trace

The trace tr(A) is the sum of diagonal elements: tr(A) = Σ aii

Key Properties

  • tr(A + B) = tr(A) + tr(B)
  • tr(cA) = c · tr(A)
  • tr(AB) = tr(BA)
  • tr(AT) = tr(A)
  • tr(A) = sum of eigenvalues

Special Cases

Matrix TypeTrace Property
Identity Intr(I) = n
Projection Ptr(P) = rank(P)
Nilpotent Ntr(N) = 0
Idempotent Etr(E) = rank(E)
