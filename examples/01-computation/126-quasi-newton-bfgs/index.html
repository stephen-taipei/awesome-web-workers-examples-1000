About BFGS Algorithm

BFGS (Broyden-Fletcher-Goldfarb-Shanno) is a quasi-Newton optimization method that approximates the inverse Hessian matrix using gradient information, avoiding the costly computation of second derivatives.

BFGS Update:

s_k = x_{k+1} - x_k           (Step)
y_k = ∇f_{k+1} - ∇f_k         (Gradient difference)
ρ_k = 1 / (y_k^T s_k)

H_{k+1} = (I - ρ_k s_k y_k^T) H_k (I - ρ_k y_k s_k^T) + ρ_k s_k s_k^T

x_{k+1} = x_k - α_k H_k ∇f_k  (Update with line search)
