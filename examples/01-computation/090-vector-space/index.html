About Vector Spaces

Given a matrix A, we can analyze the associated vector spaces: column space (range), row space, and null space (kernel).

Key Concepts

PropertyDefinitionFormula
RankDimension of column spacerank(A) = # pivot columns
NullityDimension of null spacenullity(A) = n - rank(A)
Column SpaceSpan of columnsCol(A) = {Ax : x ∈ Rⁿ}
Null SpaceSolutions to Ax = 0Null(A) = {x : Ax = 0}

Rank-Nullity Theorem

rank(A) + nullity(A) = n (number of columns)

Row Reduction (RREF)

  • Leading entry (pivot) is 1 in each nonzero row
  • Each pivot is the only nonzero in its column
  • Pivot columns indicate linearly independent columns
  • Free variables correspond to nullspace basis vectors

Applications

  • Testing linear independence
  • Finding basis for solution space
  • Image compression (low-rank approximation)
  • System solvability analysis
