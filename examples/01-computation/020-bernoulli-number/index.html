About Bernoulli Numbers

Bernoulli numbers are a sequence of rational numbers with deep connections to number theory, combinatorics, and analysis. They appear in:

  • Taylor series expansion of tan(x) and tanh(x)
  • Euler-Maclaurin summation formula
  • Sum of powers formulas
  • Riemann zeta function at even integers

Note: B(n) = 0 for odd n > 1

Algorithm: Akiyama-Tanigawa

This calculator uses the Akiyama-Tanigawa algorithm, which efficiently computes Bernoulli numbers using a triangular scheme:

  1. Initialize: A[j] = 1/(j+1) for j = 0, 1, ..., n
  2. For m = 0 to n:
    • B(m) = A[0]
    • Update: A[j] = (j+1) × (A[j] - A[j+1])

The algorithm uses exact rational arithmetic to preserve precision.
