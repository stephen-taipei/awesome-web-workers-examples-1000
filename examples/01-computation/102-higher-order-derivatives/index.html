About Higher-Order Derivatives

Higher-order derivatives measure how lower-order derivatives change. They are crucial in physics (acceleration, jerk), optimization (convexity), and Taylor series.

Central Difference Formulas

2nd Derivative
f''(x) ≈ [f(x+h) - 2f(x) + f(x-h)] / h²
O(h²)
3rd Derivative
f'''(x) ≈ [f(x+2h) - 2f(x+h) + 2f(x-h) - f(x-2h)] / 2h³
O(h²)
4th Derivative
f''''(x) ≈ [f(x+2h) - 4f(x+h) + 6f(x) - 4f(x-h) + f(x-2h)] / h⁴
O(h²)

Richardson Extrapolation

Richardson extrapolation improves accuracy by combining estimates with different step sizes:

  • Basic idea: D(h) ≈ D + c₁h² + c₂h⁴ + ...
  • Extrapolation: D* = [4D(h/2) - D(h)] / 3 gives O(h⁴)
  • Higher order: Repeated extrapolation further reduces error

Applications

  • Physics: f'' = acceleration, f''' = jerk, f'''' = snap
  • Optimization: f'' determines convexity/concavity
  • Taylor Series: All derivatives needed for approximation
  • Numerical PDEs: Higher derivatives in wave/beam equations
