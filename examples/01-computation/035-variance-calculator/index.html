About Variance

Variance measures how far a set of numbers is spread out from their average value. It's the square of standard deviation.

Formulas

Type Symbol Formula
Population σ² σ² = Σ(xᵢ - μ)² / N
Sample s² s² = Σ(xᵢ - x̄)² / (n-1)

ANOVA Components

  • SS Between: Variance between group means
  • SS Within: Variance within groups
  • F Ratio: MS Between / MS Within
  • ICC: Intraclass Correlation Coefficient

Two-Pass Algorithm

First pass calculates the mean, second pass calculates sum of squared deviations. More accurate than single-pass naive algorithm.
