About Numerical Integration

Numerical integration (quadrature) approximates definite integrals when analytical solutions are difficult or impossible.

Integration Methods

Method Formula Accuracy
Rectangle h × Σf(xᵢ) O(h) or O(h²)
Trapezoidal h/2 × [f₀ + 2Σfᵢ + fₙ] O(h²)
Simpson's 1/3 h/3 × [f₀ + 4f₁ + 2f₂ + ... + fₙ] O(h⁴)
Simpson's 3/8 3h/8 × [f₀ + 3f₁ + 3f₂ + 2f₃ + ...] O(h⁴)
Boole's Rule 2h/45 × [7f₀ + 32f₁ + 12f₂ + ...] O(h⁶)
Romberg Richardson extrapolation O(h²ᵏ)

Romberg Integration

Uses Richardson extrapolation to improve trapezoidal estimates:

R[i,j] = (4ʲR[i,j-1] - R[i-1,j-1]) / (4ʲ - 1)
