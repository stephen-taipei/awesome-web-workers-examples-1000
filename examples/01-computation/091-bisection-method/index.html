About Bisection Method

The bisection method finds a root of f(x) = 0 by repeatedly halving an interval [a, b] where f(a) and f(b) have opposite signs.

Algorithm

  1. Check that f(a) × f(b) < 0 (root exists by IVT)
  2. Compute midpoint: c = (a + b) / 2
  3. If f(c) = 0 or (b - a) / 2 < ε, return c
  4. If f(a) × f(c) < 0, set b = c; else set a = c
  5. Repeat until convergence

Properties

PropertyValue
ConvergenceGuaranteed (if f continuous)
OrderLinear (halving each step)
Error bound|eₙ| ≤ (b-a)/2^(n+1)
Iterations for εn ≈ log₂((b-a)/ε)

Advantages & Disadvantages

  • Pro: Always converges if root exists in [a,b]
  • Pro: No derivatives required
  • Pro: Predictable number of iterations
  • Con: Slower than Newton's method
  • Con: Requires bracketing interval
