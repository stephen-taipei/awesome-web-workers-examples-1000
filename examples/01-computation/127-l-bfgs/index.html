About L-BFGS

L-BFGS (Limited-memory BFGS) is a memory-efficient variant of BFGS that only stores the last m vector pairs instead of the full n×n inverse Hessian approximation. This makes it suitable for large-scale optimization.

Two-Loop Recursion:

Memory: Store m pairs of {s_k, y_k}

For k iterations:
  s_k = x_{k+1} - x_k
  y_k = ∇f_{k+1} - ∇f_k

Direction computation (two-loop):
  q = ∇f_k
  for i = k-1 to k-m:
    α_i = ρ_i * s_i^T * q
    q = q - α_i * y_i
  r = H_0 * q
  for i = k-m to k-1:
    β = ρ_i * y_i^T * r
    r = r + s_i * (α_i - β)
  p = -r  (search direction)
