Differential Evolution

Web Worker Example #132 - Evolutionary Optimization

About Differential Evolution

Differential Evolution (DE) is a population-based optimization algorithm that uses vector differences for perturbing the population. It's particularly effective for continuous parameter optimization problems.

Algorithm:

For each target vector x_i:
  1. Mutation: v = x_r1 + F × (x_r2 - x_r3)
     where r1, r2, r3 are distinct random indices ≠ i

  2. Crossover: u_j = v_j if rand < CR or j = j_rand
                u_j = x_i,j otherwise

  3. Selection: x_i = u if f(u) ≤ f(x_i)
                x_i = x_i otherwise

Strategies:
  DE/rand/1:   v = x_r1 + F(x_r2 - x_r3)
  DE/best/1:   v = x_best + F(x_r1 - x_r2)
  DE/rand/2:   v = x_r1 + F(x_r2 - x_r3) + F(x_r4 - x_r5)
  DE/best/2:   v = x_best + F(x_r1 - x_r2) + F(x_r3 - x_r4)
  DE/current-to-best/1: v = x_i + F(x_best - x_i) + F(x_r1 - x_r2)

Parameters:
  F  = Mutation factor (0.4 - 1.0)
  CR = Crossover probability (0.1 - 0.9)
  NP = Population size (5D - 10D)
