About Percentiles

A percentile indicates the value below which a given percentage of data falls.

Common Percentiles

Percentile Also Known As Meaning
P25 Q1, First Quartile 25% of data below
P50 Q2, Median 50% of data below
P75 Q3, Third Quartile 75% of data below
P90, P95, P99 Top percentiles Used for outliers, SLAs

Interpolation Methods

  • Linear: Linear interpolation between data points
  • Nearest: Returns nearest data point
  • Lower/Higher: Returns floor/ceil index value
