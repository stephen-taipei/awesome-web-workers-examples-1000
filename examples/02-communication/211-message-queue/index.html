Web Worker Example #211 - High-Throughput Ring Buffer

About Message Queue

A Message Queue with a Ring Buffer provides lock-free, high-throughput communication between workers. The circular buffer design allows efficient memory reuse without dynamic allocation.

Ring Buffer Design:

Ring Buffer (Circular Queue):

    write ──→ ┌───┬───┬───┬───┬───┬───┬───┬───┐
              │ 5 │ 6 │ 7 │   │   │ 1 │ 2 │ 3 │
              └───┴───┴───┴───┴───┴───┴───┴───┘
                              ↑           read ──→

Enqueue: buffer[write % size] = data; write++
Dequeue: data = buffer[read % size]; read++

Empty: read == write
Full:  write - read == size

SharedArrayBuffer Layout:
┌──────────────────────────────────────────┐
│ [0] read      - read position            │
│ [1] write     - write position           │
│ [2] size      - buffer capacity          │
│ [3] msgSize   - bytes per message        │
│ [4+] buffer   - message storage          │
└──────────────────────────────────────────┘

Lock-Free Algorithm:
- Single producer, single consumer (SPSC)
- Uses memory barriers (Atomics)
- No locks required for SPSC pattern
