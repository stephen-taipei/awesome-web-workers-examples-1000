支援類型一覽

✅ 支援的類型

  • 原始值 (string, number, boolean, null, undefined)
  • Date, RegExp
  • Array, Object (含巢狀)
  • Map, Set
  • ArrayBuffer, TypedArray
  • Blob, File, FileList
  • ImageData, ImageBitmap

❌ 不支援的類型

  • Function
  • Symbol
  • DOM 節點
  • Error 物件 (部分瀏覽器)
  • WeakMap, WeakSet
  • 原型鏈 (prototype)

關於此範例

  • 結構化克隆：HTML 規範定義的深層複製演算法
  • 自動處理：postMessage 自動使用結構化克隆
  • 循環引用：可正確處理對象間的循環引用
  • 限制：函數、Symbol 等無法被克隆
