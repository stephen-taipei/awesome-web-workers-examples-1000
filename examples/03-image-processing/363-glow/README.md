# 光暈效果 (Glow)

本範例展示如何使用 Web Worker 實現圖像的光暈（或稱 Bloom、泛光）效果。

## 功能簡介

- **光暈生成**：提取圖像中的高亮區域，進行模糊處理，然後疊加回原圖，模擬明亮物體發光的視覺效果。
- **參數調整**：
    -   **閾值 (Threshold)**：決定哪些亮度的區域會產生光暈。
    -   **模糊半徑 (Blur Radius)**：控制光暈擴散的範圍。
    -   **強度 (Intensity)**：控制光暈的亮度/疊加權重。

## 技術重點

1.  **高亮提取 (Highlight Extraction)**：
    -   遍歷所有像素，計算亮度（Luminance）。
    -   僅保留亮度超過閾值的像素，其餘置黑。

2.  **高斯模糊 (Gaussian Blur)**：
    -   對提取的高亮區域應用高斯模糊。
    -   使用可分離卷積核（Separable Kernel）優化性能，將二維卷積拆解為兩次一維卷積（水平與垂直）。

3.  **圖像合成 (Compositing)**：
    -   使用線性減淡（Linear Dodge / Additive）混合模式，將模糊後的高亮層疊加到原圖上。
    -   $ Final = Original + BlurredHighlights \times Intensity $

## 使用方式

1.  開啟網頁。
2.  拖放或選擇一張圖片上傳（建議使用包含明亮光源的圖片，如夜景、霓虹燈等）。
3.  調整「閾值」選取發光區域。
4.  調整「模糊半徑」設定發光範圍。
5.  調整「強度」設定發光亮度。
6.  點擊「應用效果」按鈕。
