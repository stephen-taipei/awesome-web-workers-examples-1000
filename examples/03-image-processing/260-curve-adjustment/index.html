關於此範例

  • 曲線調整：透過自訂輸入到輸出的映射曲線，調整影像的亮度、對比度和色調
  • LUT 查找表：預先計算所有 256 個可能輸入值的輸出值，實現高效處理
  • Worker 類型：Dedicated Worker (專屬 Worker)
  • 技術重點：使用 Cubic Spline 插值產生平滑曲線，支援 RGB 分通道調整
  • 效能目標：1080p 圖片處理時間 < 100ms
