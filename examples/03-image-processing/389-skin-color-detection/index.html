膚色檢測 (Skin Color Detection)

利用 Web Worker 與 YCbCr 顏色空間進行膚色區域分割

Web Worker YCbCr Intermediate

功能描述

本示例演示如何在 Web Worker 中進行膚色檢測。膚色檢測常作為人臉檢測或手勢識別的預處理步驟。本示例使用 YCbCr 顏色空間，因為在該空間中膚色聚集在一個較緊湊的區域，不受亮度 (Y) 影響太大。

Skin Region Condition (YCbCr): Cb in [77, 127] Cr in [133, 173]
