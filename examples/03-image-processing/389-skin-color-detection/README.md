# 膚色檢測 (Skin Color Detection)

本示例展示了如何使用 Web Workers 和顏色空間轉換技術來檢測圖像中的膚色區域。

## 技術重點

1.  **顏色空間轉換 (RGB to YCbCr)**:
    - RGB 顏色空間中，膚色受亮度影響較大，難以定義簡單的閾值。
    - YCbCr 空間將亮度 (Y) 與色度 (Cb, Cr) 分離。
    - 膚色在 CbCr 平面上通常聚集在一個較小的橢圓或矩形區域內。

2.  **閾值分割 (Thresholding)**:
    - 根據經驗公式設定 Cb 和 Cr 的範圍。
    - 一般範圍: `Cb ∈ [77, 127]`, `Cr ∈ [133, 173]`。

## 使用說明

1.  上傳一張包含人物的圖片。
2.  選擇**顯示模式**:
    - **Binary Mask**: 二值化圖像，白色為膚色區域。
    - **Overlay**: 在原圖上高亮顯示膚色區域，非膚色區域變暗。
    - **Extract**: 僅保留膚色區域，其餘背景變黑。
3.  點擊「開始檢測」。

## 注意事項

- 環境光線（如過黃或過藍的光源）會顯著影響檢測結果。
- 類似膚色的物體（如木頭、橙色衣服）可能會被誤檢。
- 簡單的閾值法無法處理所有種族和光照條件，更高級的方法可能需要自適應閾值或機器學習模型。
