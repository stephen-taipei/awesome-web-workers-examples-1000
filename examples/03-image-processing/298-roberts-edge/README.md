# Roberts 邊緣檢測 (Roberts Edge Detection)

這個範例展示了如何使用 Web Workers 實作 Roberts Cross 邊緣檢測算子。

## 功能簡介

Roberts Cross 算子是最早期的邊緣檢測算子之一（1963年）。它利用對角線方向的差分來近似梯度。

相比於 Sobel 和 Prewitt 的 3x3 卷積核，Roberts 算子使用 2x2 的卷積核。
優點是計算簡單，對邊緣定位非常精確（因為它只檢查 2x2 區域）。
缺點是對噪聲非常敏感，因為它沒有任何平滑或平均的機制。

## 技術重點

1.  **灰階轉換**：首先將彩色圖片轉換為灰階。
2.  **卷積運算**：使用兩個 2x2 的卷積核（對角線方向）對圖像進行掃描。
    *   Gx:
        ```
         1  0
         0 -1
        ```
    *   Gy:
        ```
         0  1
        -1  0
        ```
3.  **梯度計算**：強度 = `sqrt(Gx² + Gy²)` 或近似為 `|Gx| + |Gy|`。

## 使用方法

1.  選擇或拖放一張圖片。
2.  選擇顯示模式（強度或二值化）。
3.  如果是二值化模式，可以調整閾值。
4.  點擊「偵測邊緣」查看結果。
