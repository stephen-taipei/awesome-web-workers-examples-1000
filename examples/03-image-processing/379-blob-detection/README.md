# Blob Detection (斑點檢測)

Web Worker Example #379

## 簡介

本範例展示如何使用 Web Worker 進行斑點檢測 (Blob Detection)。斑點檢測旨在識別圖像中與周圍環境有顯著差異的區域。這裡使用的是 **LoG (Laplacian of Gaussian)** 方法，這是一種結合了高斯平滑和拉普拉斯算子的檢測技術。

## 功能

*   **LoG 濾波**: 計算高斯-拉普拉斯算子的卷積響應。
*   **尺度空間**: 支援調整 Sigma (尺度) 參數以檢測不同大小的斑點。
*   **極值檢測**: 在卷積響應圖中尋找局部極值點作為斑點中心。

## 技術重點

1.  **卷積運算**: 使用 LoG 核對圖像進行卷積，這是計算密集型的操作。
2.  **尺度歸一化**: 為了確保響應值在不同尺度下的可比性，對響應進行歸一化處理。
3.  **非極大值抑制**: 在局部鄰域內尋找響應最強的點。

## 使用方式

1.  點擊上傳區域選擇圖片。
2.  調整 **Sigma (Scale)** 以匹配目標斑點的大小。較大的 Sigma 用於檢測較大的斑點。
3.  調整 **Threshold** (閾值) 以過濾掉響應較弱的噪聲點。
4.  點擊 "Detect Blobs" 開始檢測。
5.  結果圖像會以粉紅色圓圈標記出檢測到的斑點，圓圈大小對應於檢測尺度。
