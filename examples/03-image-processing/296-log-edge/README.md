# LoG 邊緣檢測 (Laplacian of Gaussian)

這個範例展示了如何使用 Web Workers 實作 Laplacian of Gaussian (LoG) 邊緣檢測算子。

## 功能簡介

LoG 算子結合了高斯平滑（Gaussian Smoothing）和拉普拉斯算子（Laplacian Operator）。
1.  **高斯平滑**：先對圖像進行平滑處理，去除噪聲。
2.  **拉普拉斯運算**：對平滑後的圖像進行二階微分，找出邊緣（零交叉點）。

這兩個步驟可以合併為一個卷積核，形狀類似墨西哥帽（Mexican Hat），因此也常被稱為墨西哥帽小波。

## 技術重點

1.  **卷積核生成**：根據設定的 σ (Sigma) 值動態生成 LoG 卷積核。核心大小通常設為 `6σ` 左右。
2.  **零交叉檢測 (Zero Crossing)**：在二值化模式下，我們不只是簡單地看數值大小，而是尋找像素值符號發生變化的位置（正轉負或負轉正），這通常對應於圖像的邊緣中心。

## 使用方法

1.  選擇或拖放一張圖片。
2.  調整 **Sigma (σ)** 參數。較大的 σ 會產生更強的平滑效果，只檢測出主要輪廓；較小的 σ 則會保留更多細節，但也包含更多噪聲。
3.  選擇顯示模式：
    *   **灰階 (Gray)**：顯示 LoG 卷積的原始響應（加上偏移量 128）。
    *   **二值化 (Zero Crossing)**：顯示經過零交叉檢測後的二值化邊緣。
4.  如果是二值化模式，可以調整閾值來過濾強度較弱的邊緣。
5.  點擊「偵測邊緣」查看結果。
