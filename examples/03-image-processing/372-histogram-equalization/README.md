# Histogram Equalization (直方圖均衡化)

Web Worker Example #372

## 簡介

本範例展示如何使用 Web Worker 進行圖片的直方圖均衡化 (Histogram Equalization) 處理。直方圖均衡化是一種增強圖片對比度的技術，特別適用於背景和前景都太亮或太暗的圖片。

## 功能

*   **直方圖計算**: 計算圖片的亮度直方圖。
*   **均衡化**: 使用累積分布函數 (CDF) 重新映射像素亮度，使直方圖分布更均勻。
*   **HSL 色彩空間**: 為了保持色彩平衡，轉換在 HSL 色彩空間的亮度 (L) 通道上進行，而不是直接在 RGB 通道上進行。

## 技術重點

1.  **色彩空間轉換**: RGB -> HSL -> RGB 轉換。
2.  **CDF 計算**: 累積分布函數的計算與正規化。
3.  **像素映射**: 建立查找表 (LUT) 進行快速映射。
4.  **Web Worker**: 將繁重的像素處理工作移至背景執行緒，避免阻塞 UI。

## 使用方式

1.  點擊上傳區域或拖放圖片。
2.  點擊 "Apply Equalization" 按鈕。
3.  查看原始圖片與處理後圖片的對比，以及對應的直方圖。
4.  點擊 "Download Image" 下載結果。
