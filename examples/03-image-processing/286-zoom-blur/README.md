# 縮放模糊 (Zoom Blur)

本範例展示如何使用 Web Workers 實現影像的縮放模糊 (放射狀縮放) 效果。

## 功能簡介

縮放模糊模擬相機在曝光過程中變焦產生的效果，這會產生從畫面中心向外放射的線條感，常用於創造速度感或強調畫面焦點。

### 主要功能

1.  **中心控制**：可設定變焦的中心點 (X, Y 座標)。
2.  **強度控制**：可設定模糊的強度 (採樣距離)。
3.  **品質控制**：可設定採樣點數量。
4.  **即時預覽**：在原始圖片上顯示中心點位置。

## 技術實現

### 模糊算法

對於每個目標像素 `P(x, y)`：

1.  計算相對於中心點 `C(cx, cy)` 的向量 `V = P - C`。
2.  沿著該向量向中心方向進行採樣。採樣點位置為 `C + V * scale`，其中 `scale` 從 `1.0` 變化到 `1.0 - strength`。
3.  對所有採樣點的顏色進行平均。
4.  使用雙線性插值獲取非整數座標的顏色值。

### Web Worker 處理

-   **主執行緒**：負責 UI 互動與 Canvas 顯示。
-   **Worker 執行緒**：並行執行每個像素的採樣與平均計算。

## 使用方法

1.  上傳圖片或載入範例。
2.  調整「中心 X」和「中心 Y」設定焦點。
3.  調整「強度」設定動態感。
4.  點擊「套用模糊」。

## 效能考量

-   與徑向模糊類似，主要開銷在於大量的採樣與插值運算。
-   適合使用 Web Workers 避免阻塞 UI。
