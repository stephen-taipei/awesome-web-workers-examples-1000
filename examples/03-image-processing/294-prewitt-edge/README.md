# Prewitt 邊緣檢測 (Prewitt Edge Detection)

這個範例展示了如何使用 Web Workers 實作 Prewitt 邊緣檢測算子。

## 功能簡介

Prewitt 算子與 Sobel 算子非常相似，也是用於邊緣檢測的一階微分算子。主要區別在於 Prewitt 算子的卷積核權重是均勻的（全為 1 或 -1），而 Sobel 算子在中心位置有加權（2）。

因為權重均勻，Prewitt 算子對噪聲稍微敏感一些，但計算上稍微快一點點（全是加減法，沒有乘法，雖然後來硬體乘法也很快了）。

我們提供了兩種顯示模式：
1.  **梯度強度 (Magnitude)**：顯示梯度的絕對值大小。
2.  **二值化 (Binary)**：根據設定的閾值，將梯度強度轉換為黑白影像。

## 技術重點

1.  **灰階轉換**：首先將彩色圖片轉換為灰階。
2.  **卷積運算**：使用兩個 3x3 的卷積核（水平和垂直）對圖像進行掃描。
    *   Gx (水平):
        ```
        -1  0  1
        -1  0  1
        -1  0  1
        ```
    *   Gy (垂直):
        ```
        -1 -1 -1
         0  0  0
         1  1  1
        ```
3.  **梯度計算**：強度 = `sqrt(Gx² + Gy²)`

## 使用方法

1.  選擇或拖放一張圖片。
2.  選擇顯示模式。
3.  如果是二值化模式，可以調整閾值。
4.  點擊「偵測邊緣」查看結果。
