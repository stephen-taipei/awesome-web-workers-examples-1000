About Saturation Adjustment

飽和度調整改變圖像色彩的鮮豔程度。高飽和度使顏色更加鮮豔，低飽和度使顏色更加灰暗。這需要將 RGB 色彩空間轉換為 HSL（色相、飽和度、亮度）色彩空間進行處理。

HSL 色彩空間轉換：

RGB → HSL 轉換：
  max = max(R, G, B) / 255
  min = min(R, G, B) / 255
  L = (max + min) / 2

  若 max = min：S = 0, H = 0
  否則：
