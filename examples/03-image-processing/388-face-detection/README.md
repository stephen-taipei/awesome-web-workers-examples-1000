# 人臉檢測 (Face Detection)

本示例展示了如何使用 Web Workers 和積分圖 (Integral Image) 技術來實現一個簡易的人臉檢測算法。

## 技術重點

1.  **積分圖 (Integral Image)**:
    - 預先計算圖像的積分圖，使得圖像中任意矩形區域的像素和可以在 O(1) 時間內計算完成。
    - 公式: `Sum(rect) = I(D) + I(A) - I(B) - I(C)`。

2.  **特徵掃描 (Feature Scanning)**:
    - 模擬 Viola-Jones 算法的級聯分類器思想。
    - 使用滑動窗口在不同尺度下掃描圖像。
    - 應用基於亮度的簡易 Haar-like 特徵規則（例如：眼睛區域比前額和臉頰暗）。

3.  **非極大值抑制 (NMS)**:
    - 合併重疊的檢測框，輸出最終的人臉位置。

## 使用說明

1.  點擊上傳區域或拖曳圖片上傳（建議使用正面清晰的人臉照片）。
2.  調整搜尋參數（可選）：
    - **Min Scale**: 最小搜尋窗口比例。
    - **Max Scale**: 最大搜尋窗口比例。
    - **Scale Step**: 搜尋窗口放大的倍率步長。
3.  點擊「開始檢測」按鈕。
4.  結果將顯示在右側畫布上，人臉區域會被綠色方框標示。

## 注意事項

- 本示例為純 JavaScript 實現的簡易版本，未加載預訓練的權重文件 (XML/JSON)，因此準確率與 OpenCV 等專業庫有差距。
- 主要目的是演示如何在 Web Worker 中高效處理像素數據和實現滑動窗口算法。
