人臉檢測 (Face Detection)

利用 Web Worker 與 Integral Image 進行簡易人臉檢測

Web Worker OffscreenCanvas Advanced

功能描述

本示例演示如何在 Web Worker 中實現一個簡易的人臉檢測算法。由於完整的 Viola-Jones 算法需要大量的訓練數據，這裡演示其核心概念：積分圖 (Integral Image) 計算與特徵掃描。此示例使用簡化的特徵規則來檢測類人臉區域（註：僅為演示原理，準確率不如 OpenCV 等成熟庫）。

Integral Image I(x,y) = Σ i(x',y') for x'≤x, y'≤y RectSum(r) = I(D) + I(A) - I(B) - I(C)
