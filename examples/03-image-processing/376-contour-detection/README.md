# Contour Detection (輪廓檢測)

Web Worker Example #376

## 簡介

本範例展示如何使用 Web Worker 進行輪廓檢測。輪廓檢測是影像處理中的重要步驟，用於分離物體、識別形狀或計算物體數量。

## 功能

*   **二值化**: 根據設定的閾值將圖片轉換為黑白二值圖像。
*   **輪廓追蹤**: 使用 Moore-Neighbor Tracing 算法追蹤二值圖像中的邊界。
*   **視覺化**: 將檢測到的輪廓以不同顏色繪製在原始圖像或黑色背景上。

## 技術重點

1.  **二值化處理**: 預處理步驟，將灰階圖像轉換為 0 和 1。
2.  **Moore-Neighbor Tracing**: 一種邊界追蹤算法，通過順時針掃描鄰居來找到下一個邊界像素。
3.  **狀態管理**: 記錄已訪問的像素以避免重複追蹤同一輪廓。

## 使用方式

1.  點擊上傳區域選擇圖片。
2.  調整 **Binarization Threshold** (二值化閾值) 以獲得最佳的黑白分離效果。
3.  選擇 **Invert Input** (反轉輸入) 如果你要在亮背景上找暗物體。
4.  點擊 "Detect Contours" 開始檢測。
