# Double Exposure

此示例展示了如何使用 Web Worker 實現雙重曝光（Double Exposure）效果。

## 技術原理

雙重曝光原指在膠卷攝影中，不捲動底片而進行兩次曝光，將兩個畫面重疊在一起。數位圖像處理中，這通常通過「混合模式（Blend Mode）」來實現。

主要步驟：
1.  **圖像調整**：將「疊加圖（Overlay）」調整大小以匹配「底圖（Base）」的尺寸，通常使用 `Cover` 模式（裁剪以填滿）。
2.  **像素混合**：遍歷每個像素，根據選擇的混合模式計算新的顏色值。
3.  **透明度混合**：將混合後的結果與底圖根據「強度（Opacity）」進行插值。

## 混合模式說明

*   **濾色 (Screen)**：將兩張圖的顏色反轉相乘再反轉。結果總是比較亮。適合黑色背景的素材。
*   **色彩增值 (Multiply)**：將兩張圖的顏色相乘。結果總是比較暗。適合白色背景的素材。
*   **覆蓋 (Overlay)**：結合 Multiply 和 Screen。若底圖顏色暗則 Multiply，亮則 Screen。增強對比度。
*   **變亮 (Lighten)**：取兩張圖中較亮的像素值。
*   **變暗 (Darken)**：取兩張圖中較暗的像素值。

## 使用方式

1.  上傳底圖（通常是人像或主體）。
2.  上傳疊加圖（通常是風景、城市或紋理）。
3.  選擇合適的混合模式和強度。建議嘗試 `Screen` 或 `Lighten` 模式來保留人像的輪廓（如果人像背景較暗）。
