# 文字區域檢測 (Text Region Detection)

本示例展示了如何使用 Web Workers 和邊緣密度分析技術來檢測圖像中的文字區域。

## 技術重點

1.  **邊緣檢測 (Edge Detection)**:
    - 使用 Sobel 算子計算圖像的邊緣強度。
    - 文字區域通常包含密集的邊緣變化。

2.  **形態學膨脹 (Morphological Dilation)**:
    - 使用較寬的卷積核（例如 11x3）對邊緣圖像進行膨脹操作。
    - 目的是將相鄰的字符邊緣連接成一個完整的單詞或文本行區域。

3.  **連通區域分析 (Connected Components)**:
    - 尋找膨脹後的白色連通區域。
    - 根據邊界框的幾何特徵（如面積、長寬比）過濾非文字區域。

## 使用說明

1.  上傳包含文字的圖片（如文件掃描件、路牌）。
2.  點擊「開始檢測」。
3.  系統將在右側標示出檢測到的文字區域（橙色方框）。

## 注意事項

- 本算法基於邊緣密度，因此可能會錯誤檢測到其他高紋理區域（如樹葉、柵欄）。
- 對於極小或極大的文字，或者背景複雜的文字，檢測效果可能受限。
- 更高級的場景文字檢測（Scene Text Detection）通常需要使用深度學習模型（如 EAST, CRAFT）。
