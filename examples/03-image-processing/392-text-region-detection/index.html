文字區域檢測 (Text Region Detection)

利用 Web Worker 與邊緣密度分析來定位文字區塊

Web Worker Edge Density Advanced

功能描述

本示例演示如何檢測圖片中的文字區域。文字通常具有高密度的邊緣信息。通過檢測邊緣（Sobel），然後進行形態學膨脹 (Dilation) 將相鄰字符連接成行，最後通過輪廓分析找到可能的文字區塊。

Edges = Sobel(Gray) TextRegion = Dilate(Edges) -> FindContours
