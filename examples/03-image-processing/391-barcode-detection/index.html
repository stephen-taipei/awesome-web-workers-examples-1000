條碼檢測 (Barcode Detection)

利用 Web Worker 與梯度分析進行條碼區域定位

Web Worker Gradient Analysis Intermediate

功能描述

本示例演示如何檢測圖片中的一維條碼位置。條碼的特徵是具有大量的垂直邊緣。我們可以通過計算水平梯度減去垂直梯度，並進行形態學操作（模糊和二值化）來突出顯示條碼區域。

Gradient = |SobelX| - |SobelY| Enhanced = Blur(Gradient) -> Threshold -> Close
