# 條碼檢測 (Barcode Detection)

本示例展示了如何使用 Web Workers 和形態學圖像處理技術來檢測一維條碼區域。

## 技術重點

1.  **梯度計算 (Gradient Computation)**:
    - 條碼由黑白條紋組成，具有極強的垂直梯度和較弱的水平梯度。
    - 使用 Sobel 算子計算水平梯度 (Gx) 和垂直梯度 (Gy)。
    - 計算 `|Gx| - |Gy|` 以突出顯示垂直紋理區域。

2.  **形態學操作 (Morphological Operations)**:
    - **模糊 (Blur)**: 平滑噪聲。
    - **二值化 (Thresholding)**: 將高梯度區域分離出來。
    - **閉運算 (Closing)**: 先膨脹後腐蝕。使用寬矩形核 (Kernel) 將離散的條紋連接成一個完整的實心矩形塊。

3.  **連通區域分析 (Connected Components)**:
    - 尋找二值圖像中的連通白色區域。
    - 根據面積和長寬比過濾非條碼區域，輸出條碼的邊界框。

## 使用說明

1.  上傳包含一維條碼的圖片（如商品條碼）。
2.  點擊「開始檢測」。
3.  系統將在右側標示出檢測到的條碼區域（藍色方框）。

## 注意事項

- 本算法針對水平放置的條碼進行優化，對於傾斜或垂直放置的條碼效果可能不佳。
- 複雜背景或具有類似紋理（如百葉窗、柵欄）的物體可能會造成誤檢。
