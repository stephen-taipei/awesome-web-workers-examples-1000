# ASCII Art

此示例展示了如何使用 Web Worker 將圖片轉換為 ASCII 藝術。

## 技術原理

1.  **縮放與採樣**：
    *   首先將圖片縮小到目標網格大小（例如 150 列寬）。
    *   計算目標高度時考慮字元的寬高比（通常字元比像素高，寬高比約 0.5-0.6），以防止圖片變形。
    *   使用 Canvas API 進行高效的縮放和像素數據提取。

2.  **亮度映射**：
    *   計算每個像素的亮度（平均值或加權亮度）。
    *   將亮度值（0-255）映射到預定義的字元集索引。

3.  **字元集 (Charset)**：
    *   不同的字元擁有不同的視覺密度。
    *   例如：`@` 佔用像素多，看起來較亮（在黑底白字終端中）或較暗（在白底黑字紙張上）。
    *   本示例預設為黑底白字，因此密度高的字元代表亮部。

## 參數說明

*   **字元集**：選擇不同的字元組合以獲得不同的視覺風格。
*   **寬度**：控制輸出 ASCII 圖形的水平字元數。
*   **反轉亮度**：反轉映射關係（亮變暗，暗變亮）。

## 使用建議

*   對於細節豐富的圖片，增加「寬度」可以獲得更清晰的結果，但會產生非常大的文本塊。
*   建議縮小瀏覽器字體大小或在寬螢幕上查看。
