About Warm Tone Filter

The warm tone filter applies an orange/golden color cast to images, creating a cozy, sunset-like atmosphere. This effect enhances red and yellow tones while slightly reducing blue, commonly used for golden hour photography and creating inviting, nostalgic moods.

Warm Tone Transform:

For each pixel (R, G, B):

// Boost red, slightly increase green, reduce blue
newR = R × (1 + intensity × 0.3) + intensity × 15
newG = G × (1 + intensity × 0.1) + intensity × 5
newB = B × (1 - intensity × 0.2)

Alternative Matrix Form (with color enhancement):
┌───────┐   ┌──────────────────────────────┐   ┌───┐
│ newR  │   │ 1.30   0.05   0.00   15 │   │ R │
│ newG  │ = │ 0.05   1.10   0.00   5  │ × │ G │
│ newB  │   │ 0.00   0.00   0.80   0  │   │ B │
│   1   │   │  0      0      0     1  │   │ 1 │
└───────┘   └──────────────────────────────┘   └───┘

Values are clamped to [0, 255]
