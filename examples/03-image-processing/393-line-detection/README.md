# 線段檢測 (Line Detection)

本示例展示了如何使用 Web Workers 和標準霍夫變換 (Standard Hough Transform) 來檢測圖像中的直線。

## 技術重點

1.  **邊緣檢測 (Edge Detection)**:
    - 使用 Sobel 算子計算梯度，並提取強邊緣點。
    - 這些點將作為霍夫變換的輸入。

2.  **霍夫變換 (Hough Transform)**:
    - 將圖像空間 (x, y) 中的每個邊緣點映射到參數空間 (ρ, θ)。
    - 對於每個邊緣點，計算通過該點的所有可能直線，並在累加器 (Accumulator) 中對應的 (ρ, θ) 位置投票。
    - 直線方程: `ρ = x * cos(θ) + y * sin(θ)`。

3.  **峰值檢測 (Peak Detection)**:
    - 累加器中票數超過閾值的局部極大值點即對應於圖像中的一條直線。

## 使用說明

1.  上傳包含直線特徵的圖片（如建築物、幾何圖形）。
2.  調整 **Votes Threshold**：
    - 閾值越高，檢測到的直線越少但越顯著（長且連續）。
    - 閾值越低，檢測到的直線越多，但也可能包含噪聲。
3.  點擊「開始檢測」。
4.  結果將在圖像上繪製出檢測到的紅色直線。

## 注意事項

- 計算量與邊緣點數量和參數空間的分辨率成正比。
- 本示例為了演示原理，未進行複雜的優化，對於大圖片可能需要較長處理時間。
- 繪圖時將 ρ, θ 轉換回笛卡爾坐標系繪製無限長直線（在畫布範圍內截斷）。
