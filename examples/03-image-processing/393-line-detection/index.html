線段檢測 (Line Detection)

利用 Web Worker 與霍夫變換 (Hough Transform) 檢測直線

Web Worker Hough Transform Advanced

功能描述

本示例演示如何使用霍夫變換檢測圖片中的直線。首先對圖像進行邊緣檢測（Sobel + 閾值），然後將邊緣點映射到霍夫空間（參數空間 ρ-θ），通過尋找累加器中的峰值來確定直線的位置。

ρ = x * cos(θ) + y * sin(θ)
