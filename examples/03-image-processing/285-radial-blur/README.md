# 徑向模糊 (Radial Blur)

本範例展示如何使用 Web Workers 實現影像的徑向模糊 (旋轉模糊) 效果。

## 功能簡介

徑向模糊模擬物體圍繞一個中心點快速旋轉的效果。與標準的卷積濾鏡不同，徑向模糊是一種幾何變換處理，每個像素的採樣路徑取決於其相對於中心點的位置。

### 主要功能

1.  **中心控制**：可設定旋轉的中心點 (X, Y 座標)。
2.  **強度控制**：可設定旋轉的角度範圍。
3.  **品質控制**：可設定採樣點數量，採樣數越高，模糊越平滑但計算越慢。
4.  **即時預覽**：在原始圖片上顯示中心點位置。

## 技術實現

### 模糊算法

對於每個目標像素 `P(x, y)`：

1.  計算相對於中心點 `C(cx, cy)` 的半徑 `r` 和角度 `θ`。
2.  在角度範圍 `[θ - strength/2, θ + strength/2]` 內均勻選取 `N` 個採樣角度。
3.  對於每個採樣角度，計算對應的座標 `(sx, sy)`。
4.  使用雙線性插值 (Bilinear Interpolation) 獲取 `(sx, sy)` 處的像素顏色。
5.  將所有採樣點的顏色取平均，作為 `P` 的最終顏色。

### Web Worker 處理

-   **主執行緒**：負責 UI 互動、參數傳遞、以及在 Canvas 上繪製中心指示器。
-   **Worker 執行緒**：
    1.  接收圖片數據與參數。
    2.  對每個像素執行幾何採樣與顏色平均運算。
    3.  使用雙線性插值確保採樣品質。
    4.  回傳處理後的結果。

## 使用方法

1.  上傳圖片或載入範例。
2.  調整「中心 X」和「中心 Y」來改變旋轉軸心。
3.  調整「強度」來改變模糊程度。
4.  調整「品質/採樣數」來平衡速度與畫質。
5.  點擊「套用模糊」。

## 效能考量

-   計算複雜度約為 `O(Width * Height * Samples)`。
-   因為記憶體存取模式不連續 (隨機存取不同位置的像素)，快取命中率較卷積運算低。
-   雙線性插值增加了額外的浮點運算開銷，但對畫質至關重要。
