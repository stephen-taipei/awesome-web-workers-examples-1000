關於此範例

  • 選擇性調色：只對圖片中特定顏色範圍的像素進行調整
  • HSL 色彩空間：使用色相、飽和度、亮度定義顏色，更符合人眼感知
  • 顏色遮罩：根據容差計算每個像素的匹配程度，實現平滑過渡
  • Worker 類型：Dedicated Worker (專屬 Worker)
  • 效能目標：1080p 圖片處理時間 < 200ms
