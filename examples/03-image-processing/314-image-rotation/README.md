# 圖片旋轉 (Image Rotation)

圖片旋轉是基礎的幾何變換。本示例演示了如何對圖片進行任意角度的旋轉，並自動調整畫布大小以容納旋轉後的圖像。

## 原理

1.  **旋轉矩陣**：使用 2D 仿射變換矩陣進行座標映射。
2.  **反向映射 (Inverse Mapping)**：為了避免目標圖像中出現空洞，我們遍歷目標圖像的每個像素，將其座標逆向旋轉回源圖像的座標空間，然後採樣顏色。
3.  **邊界框計算**：計算源圖像四個角旋轉後的位置，以確定目標圖像所需的最小尺寸。
4.  **插值**：
    *   **最近鄰**：簡單快速，但邊緣有鋸齒。
    *   **雙線性**：平滑邊緣，效果更好。

## 使用方式

1.  上傳圖片。
2.  拖動滑桿設定「旋轉角度」。
3.  選擇「插值方法」。
4.  點擊「旋轉」。

## 技術實現

*   **Web Worker**：雖然 Canvas API 的 `rotate()` 可以快速繪製，但那只是改變了繪製上下文，並沒有生成新的像素數據用於進一步處理（或者需要 `getImageData` 開銷）。本示例直接在 Worker 中計算像素數據，展示了幾何變換的底層實現。
