About Invert Color Filter

反色濾鏡將每個像素的顏色值反轉，創造出底片效果。這是最基本的影像處理操作之一，通過 Web Worker 在背景執行緒處理大量像素數據。

反色運算公式：

對於每個像素的 RGB 值：
  R' = 255 - R
  G' = 255 - G
  B' = 255 - B
  A' = A  (保持透明度不變)

其中：
  - (R, G, B, A) 為原始像素值
  - (R', G', B', A') 為反轉後的像素值
  - 每個通道值範圍：0 ~ 255
