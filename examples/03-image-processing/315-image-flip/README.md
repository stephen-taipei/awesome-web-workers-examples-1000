# 圖片翻轉 (Image Flip)

圖片翻轉是將圖片沿著水平軸、垂直軸或兩者同時進行鏡像的操作。

## 原理

*   **水平翻轉 (Horizontal Flip)**：
    `x' = width - 1 - x`
    `y' = y`
*   **垂直翻轉 (Vertical Flip)**：
    `x' = x`
    `y' = height - 1 - y`
*   **雙向翻轉 (Both)**：
    `x' = width - 1 - x`
    `y' = height - 1 - y`
    (這相當於旋轉 180 度)

## 使用方式

1.  上傳圖片。
2.  選擇「翻轉方向」。
3.  點擊「翻轉」。

## 技術實現

*   **Web Worker**：雖然 CSS `transform: scaleX(-1)` 可以實現視覺上的翻轉，但那只是渲染層的效果。如果需要對翻轉後的像素數據進行進一步處理（如保存文件、分析特徵），則需要像本示例一樣在內存中實際重新排列像素。
