Lanczos 縮放 (Lanczos Scaling)

High Quality Image Resampling

Web Worker Image Processing Resampling

功能描述

使用 Lanczos 插值算法進行高品質圖片縮放。Lanczos 濾波器是一種基於 Sinc 函數的重採樣方法，能提供比雙線性插值更銳利和清晰的結果，但也需要更多的計算量。

L(x) = sinc(x) * sinc(x/a) if |x| < a, else 0 Kernel size determined by parameter 'a' (usually 2 or 3).
