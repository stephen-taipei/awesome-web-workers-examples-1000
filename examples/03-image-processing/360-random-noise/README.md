# 隨機噪點 (Random Noise)

本範例展示如何使用 Web Worker 快速生成隨機噪點（White Noise）。

## 功能簡介

- **噪點生成**：
    -   **單色噪點 (Monochrome)**：灰階噪點，常用於模擬電視雪花等效果。
    -   **彩色噪點 (Color)**：RGB 通道獨立隨機，生成彩色雜訊。
- **Web Worker 優勢**：生成大尺寸（如 4K）噪點圖時，涉及千萬次 `Math.random()` 調用，使用 Worker 可避免阻塞 UI。

## 使用方式

1.  開啟網頁。
2.  調整「寬度」和「高度」設定圖片尺寸。
3.  選擇「類型」（單色或彩色）。
4.  點擊「生成噪點」按鈕。
