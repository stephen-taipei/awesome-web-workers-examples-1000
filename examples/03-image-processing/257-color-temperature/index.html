Web Worker Example #257 - RGB Channel Adjustment

About Color Temperature

Color temperature adjustment simulates different lighting conditions by modifying the RGB channel balance. Warmer temperatures add yellow/orange tones, while cooler temperatures add blue tones.

Color Temperature Model:

Temperature in Kelvin (K):
  Warm light:  2700K - 3500K (candle, sunset)
  Neutral:     5000K - 5500K (daylight)
  Cool light:  6500K - 10000K (overcast, shade)

RGB Adjustment Algorithm:
  For temperature T (in Kelvin):

  If T < 6600:
    R = 255
    G = 99.4708 × ln(T/100) - 161.1196
    B = 0 (if T ≤ 1900)
      = 138.5177 × ln(T/100 - 10) - 305.0448

  If T ≥ 6600:
    R = 329.6987 × ((T/100) - 60)^(-0.1332)
    G = 288.1221 × ((T/100) - 60)^(-0.0755)
    B = 255

Temperature Shift:
  R' = R × (1 + shift × warmFactor)
  G' = G × (1 + shift × neutralFactor)
  B' = B × (1 - shift × coolFactor)

Where shift is the user-defined adjustment value
