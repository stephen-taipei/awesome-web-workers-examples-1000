About Brightness Adjustment

亮度調整是影像處理中最基本的操作之一。透過對每個像素的 RGB 值進行加減運算，可以使圖片整體變亮或變暗。使用 Web Worker 可以在不阻塞主執行緒的情況下處理大量像素數據。

亮度調整公式：

對於每個像素的 RGB 值：
  R' = clamp(R + brightness, 0, 255)
  G' = clamp(G + brightness, 0, 255)
  B' = clamp(B + brightness, 0, 255)
  A' = A  (保持透明度不變)

其中：
  - brightness: 亮度調整值 (-255 ~ +255)
  - clamp(x, min, max): 將 x 限制在 [min, max] 範圍內
  - 正值使圖片變亮，負值使圖片變暗
