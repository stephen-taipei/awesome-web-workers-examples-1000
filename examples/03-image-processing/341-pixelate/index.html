About Pixelate Effect

Create a retro "8-bit" or mosaic look by grouping pixels into blocks and averaging their colors. This effect reduces the resolution of the image details without changing the image dimensions.

Algorithm:

For each block of size N x N:
  1. Calculate average R, G, B of all pixels in the block.
  2. Fill the entire block with the average color.
