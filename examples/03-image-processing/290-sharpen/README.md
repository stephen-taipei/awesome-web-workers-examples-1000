# 影像銳化 (Sharpen)

本範例展示如何使用 Web Workers 實現影像的銳化效果。

## 功能簡介

銳化濾波器能夠增強影像的邊緣對比度，使影像看起來更清晰。這對於改善模糊的照片或增強細節非常有用。

### 主要功能

1.  **強度控制**：可設定銳化的強度。強度越高，邊緣對比越強，但也可能引入噪聲或光暈偽影。
2.  **核心預覽**：即時預覽用於銳化的 3x3 卷積核。
3.  **並行處理**：使用 Web Workers 進行卷積運算。

## 技術實現

### 卷積算法

銳化通常使用高通濾波器或拉普拉斯算子來實現。最簡單的形式是將原始影像減去其模糊版本（即非銳化遮罩原理），或者直接使用一個特定的卷積核。

本範例使用一個可調節強度的 3x3 卷積核：

```
 0  -k   0
-k 1+4k -k
 0  -k   0
```

其中 `k` 是與銳化強度成正比的係數。
- 中心像素具有正權重 `1+4k`。
- 上下左右相鄰像素具有負權重 `-k`。
- 所有權重之和為 1，保證圖像的平均亮度不變。

### Web Worker 處理

-   **主執行緒**：負責 UI 與 Canvas 顯示。
-   **Worker 執行緒**：
    1.  根據強度生成卷積核。
    2.  對影像執行卷積運算。

## 使用方法

1.  上傳圖片或載入範例。
2.  調整「銳化強度」。
3.  點擊「套用銳化」。

## 效能考量

-   3x3 卷積的計算量相對較小 (`9 * Width * Height`)，但在大圖上仍然需要一定時間。
-   使用 Web Workers 可以確保 UI 在處理過程中保持響應。
