關於影像銳化 (Sharpen)

銳化濾波器通過增強相鄰像素之間的對比度來使邊緣更加清晰。它通常使用一個中心為正值、周圍為負值的卷積核來實現。


  • 原理：銳化本質上是加強影像的高頻部分（邊緣）。這可以看作是：原圖 + (原圖 - 模糊圖) * 強度，或者直接使用拉普拉斯算子相關的卷積核。
  • 卷積核：典型的 3x3 銳化核如下： 
     0  -k   0
-k 1+4k -k
 0  -k   0
    其中 k 與強度有關。
