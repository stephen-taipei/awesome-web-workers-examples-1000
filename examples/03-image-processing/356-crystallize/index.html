晶格化 (Crystallize)

此範例展示如何使用 Voronoi 圖算法創建晶格化藝術效果。Web Worker 負責計算每個像素最近的特徵點，並將該像素顏色設定為特徵點所在區域的顏色。

Feature Points: P = {p_1, p_2, ..., p_n} For each pixel (x, y): Find k such that distance((x,y), p_k) is minimized. Color(x, y) = Color(p_k)
