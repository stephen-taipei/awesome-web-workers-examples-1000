About Contrast Adjustment

對比度調整改變圖像中最亮和最暗區域之間的差異。高對比度使圖像更加鮮明，低對比度使圖像更加柔和。透過線性映射公式，我們可以精確控制對比度的強度。

對比度調整公式：

對於每個像素的 RGB 值：
  factor = (259 × (contrast + 255)) / (255 × (259 - contrast))

  R' = clamp(factor × (R - 128) + 128, 0, 255)
  G' = clamp(factor × (G - 128) + 128, 0, 255)
  B' = clamp(factor × (B - 128) + 128, 0, 255)
  A' = A  (保持透明度不變)

其中：
  - contrast: 對比度調整值 (-255 ~ +255)
  - factor: 對比度因子（自動計算）
  - 128 為中間灰度值，作為對比度調整的基準點
  - 正值增加對比度，負值減少對比度
