圓形檢測 (Circle Detection)

利用 Web Worker 與霍夫圓變換 (Hough Circle Transform) 檢測圓形

Web Worker Hough Circle Advanced

功能描述

本示例演示如何使用霍夫圓變換檢測圖片中的圓形。為了提高效率，算法通常分為兩步：首先檢測圓心（使用梯度方向信息），然後確定半徑。

(x - a)² + (y - b)² = r² a = x - r * cos(θ) b = y - r * sin(θ)
