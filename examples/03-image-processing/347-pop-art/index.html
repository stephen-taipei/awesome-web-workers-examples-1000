About Pop Art Effect

Create a high-contrast Pop Art style image inspired by Andy Warhol's silk screen prints. The image is thresholded into distinct brightness zones and recolored.

Processing Pipeline:

  1. Grayscale & Contrast: Normalize image brightness.
  2. Thresholding: Classify pixels into 3 levels: Dark, Mid, Light.
  3. Recoloring: Map each level to a user-selected color palette.
