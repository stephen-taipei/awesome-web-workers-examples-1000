# Sobel 邊緣檢測 (Sobel Edge Detection)

這個範例展示了如何使用 Web Workers 實作 Sobel 邊緣檢測算子。

## 功能簡介

Sobel 算子是一種用於邊緣檢測的離散微分算子，它結合了高斯平滑和微分求導。算子計算圖像亮度的近似梯度，可以檢測出圖像的邊緣。

我們提供了三種顯示模式：
1.  **梯度強度 (Magnitude)**：顯示梯度的絕對值大小，邊緣處較亮。
2.  **二值化 (Binary)**：根據設定的閾值，將梯度強度轉換為黑白影像，清晰顯示邊緣。
3.  **方向 (Direction)**：使用顏色編碼顯示邊緣的方向（角度）。

## 技術重點

1.  **灰階轉換**：首先將彩色圖片轉換為灰階，簡化計算。
2.  **卷積運算**：使用兩個 3x3 的卷積核（水平和垂直）對圖像進行掃描。
    *   Gx (水平):
        ```
        -1  0  1
        -2  0  2
        -1  0  1
        ```
    *   Gy (垂直):
        ```
        -1 -2 -1
         0  0  0
         1  2  1
        ```
3.  **梯度計算**：
    *   強度 = `sqrt(Gx² + Gy²)`
    *   方向 = `atan2(Gy, Gx)`

## 使用方法

1.  選擇或拖放一張圖片。
2.  選擇顯示模式（強度、二值化或方向）。
3.  如果是二值化模式，可以調整閾值來過濾雜訊。
4.  點擊「偵測邊緣」查看結果。
