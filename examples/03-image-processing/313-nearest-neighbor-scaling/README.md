# 最近鄰縮放 (Nearest Neighbor Scaling)

最近鄰縮放是最簡單和最快的圖像縮放算法。它將目標圖像中的每個像素映射到源圖像中最近的像素，不進行任何插值或平均。

## 原理

對於目標圖像中的每個像素座標 `(x, y)`，其在源圖像中的對應座標為 `(x/scale, y/scale)`。最近鄰算法直接選取最接近該浮點座標的整數座標處的像素值。

公式：
`P_dst(x, y) = P_src( round(x/scale), round(y/scale) )`

## 特點

*   **優點**：
    *   計算速度極快。
    *   不產生模糊。
    *   完好保留原圖的顏色值（不會產生新的中間色）。
    *   非常適合**像素藝術 (Pixel Art)** 或截圖的放大。
*   **缺點**：
    *   會產生明顯的鋸齒和塊狀效應（馬賽克）。
    *   在縮小圖片時會丟失大量細節（因為只採樣了一個像素）。

## 使用方式

1.  上傳圖片（推薦像素風格圖片）。
2.  調整「縮放比例」。
3.  點擊「開始縮放」。

## 技術實現

*   **Web Worker**：實現簡單的像素映射循環。雖然簡單，但對於大圖或在 Worker 中統一處理所有圖像操作來說，這是一個很好的基礎示例。
*   **CSS Image Rendering**：在顯示結果的 Canvas 上設置了 `image-rendering: pixelated`，以確保瀏覽器在繪製 Canvas 到螢幕時也不會進行模糊插值，從而呈現清晰的像素格。
