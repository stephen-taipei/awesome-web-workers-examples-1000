# QR 碼檢測 (QR Code Detection)

本示例展示了如何使用 Web Workers 來檢測圖像中的 QR 碼定位圖案 (Finder Patterns)。

## 技術重點

1.  **二值化 (Binarization)**:
    - 將彩色圖像轉換為灰階，並使用全局平均閾值進行二值化，將像素分類為黑色或白色。

2.  **定位圖案掃描 (Finder Pattern Scanning)**:
    - QR 碼的三個角有明顯的回字形圖案。
    - 通過中心線掃描，該圖案的黑白比例應接近 `1:1:3:1:1` (黑:白:黑:白:黑)。
    - 本示例首先進行水平掃描，發現候選點後，再進行垂直方向驗證。

3.  **中心驗證 (Center Verification)**:
    - 確保候選點在水平和垂直方向都符合比例特徵。
    - 使用簡單的聚類算法合併相近的檢測點。

## 使用說明

1.  上傳包含 QR 碼的圖片。
2.  點擊「開始檢測」。
3.  系統將標示出檢測到的定位圖案中心（紅點與紅框），若檢測到三個圖案，將嘗試連接它們（綠線）。

## 注意事項

- 本算法是簡化的定位器，未包含 QR 碼解碼功能。
- 對於旋轉角度較大或變形嚴重的 QR 碼，垂直掃描驗證可能會失敗。
- 完整的 QR 碼識別庫（如 jsQR）包含更複雜的對齊圖案搜尋和透視變換處理。
