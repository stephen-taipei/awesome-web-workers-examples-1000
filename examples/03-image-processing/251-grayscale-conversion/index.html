About Grayscale Conversion

Convert color images to grayscale using weighted RGB averaging. This technique accounts for human perception of different colors, where green appears brightest and blue appears darkest.

Grayscale Conversion Formulas:

Standard (Luminosity) Method - ITU-R BT.709:
  Gray = 0.2126 Ã— R + 0.7152 Ã— G + 0.0722 Ã— B

Average Method:
  Gray = (R + G + B) / 3

Lightness Method:
  Gray = (max(R,G,B) + min(R,G,B)) / 2

Desaturation Method:
  Gray = (max(R,G,B) + min(R,G,B)) / 2

Custom Weighted:
  Gray = Wr Ã— R + Wg Ã— G + Wb Ã— B
  Where Wr + Wg + Wb = 1
