# 圖片縮放 (Image Scaling)

圖片縮放是改變圖像尺寸的基本幾何變換。本示例實現了 **雙線性插值 (Bilinear Interpolation)** 算法進行圖像縮放。

## 原理

當我們將圖片放大或縮小時，目標像素點通常對應於原圖中的非整數座標。

*   **最近鄰插值 (Nearest Neighbor)**：直接取最近的一個整數像素點的值。速度最快，但會產生鋸齒（馬賽克）。
*   **雙線性插值 (Bilinear Interpolation)**：取周圍 4 個像素點（左上、右上、左下、右下），根據距離進行加權平均。結果比最近鄰平滑，但計算量稍大。

公式：
`P(x,y) = w1*P1 + w2*P2 + w3*P3 + w4*P4`
其中 P1, P2, P3, P4 是原圖中對應浮點座標周圍的四個像素。

## 使用方式

1.  上傳圖片。
2.  調整「縮放比例」滑桿（0.1x 到 4.0x）。
3.  預覽目標尺寸。
4.  點擊「開始縮放」。

## 技術實現

*   **Web Worker**：雖然現代 Canvas 的 `drawImage` 自帶縮放且通常有硬體加速，但本示例旨在演示如何在 Worker 中用純 JavaScript 實現像素級操作。對於非常大的圖片或需要自定義高品質插值算法（如 Lanczos）的場景，這種手動實現是必要的基礎。
*   **像素映射**：通過計算 `xRatio` 和 `yRatio` 將目標座標映射回源座標空間。
