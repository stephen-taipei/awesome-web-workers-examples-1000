About Pinch Distortion

Apply a pinch effect to the image, squeezing content towards a center point. This geometric transformation modifies the radial distance of pixels relative to a center point.

Pinch Formula:

For each pixel (x, y), calculate distance r from center (cx, cy).
If r < Radius:
  Normalized dist d = r / Radius
  Factor = 1 + Strength * (1 - d)
  (u, v) = (cx, cy) + (x - cx, y - cy) * Factor

Sample source image at (u, v) using bilinear interpolation.
