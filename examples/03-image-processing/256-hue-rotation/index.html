About Hue Rotation

Hue rotation shifts all colors in an image around the color wheel by a specified angle. This technique adjusts the H (Hue) component in HSL color space while preserving saturation and lightness.

HSL Color Model:

RGB to HSL Conversion:
  M = max(R, G, B)
  m = min(R, G, B)
  L = (M + m) / 2

  S = 0                    if M = m
    = (M - m) / (1 - |2L - 1|)  otherwise

  H = 0                           if M = m
    = 60° × ((G - B) / (M - m))       if M = R
    = 60° × ((B - R) / (M - m) + 2)   if M = G
    = 60° × ((R - G) / (M - m) + 4)   if M = B

Hue Rotation:
  H' = (H + θ) mod 360°

Where θ is the rotation angle in degrees
