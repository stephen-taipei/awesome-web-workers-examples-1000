# Motion Simulation

此示例展示了如何使用 Web Worker 模擬動態模糊（Motion Blur）效果。

## 技術原理

動態模糊是當物體或相機在曝光期間移動時產生的模糊條紋。在影像處理中，這是通過沿著特定方向（運動向量）對像素進行平均化來實現的。

算法步驟：
1.  **確定運動向量**：根據使用者設定的「角度 (Angle)」和「距離 (Distance)」，計算 X 和 Y 方向的位移分量。
2.  **線積分卷積 (Line Integral Convolution)**：對於圖像中的每個像素，沿著運動向量定義的一條線段進行採樣。
3.  **平均化**：將採樣到的像素顏色值進行平均，作為該像素的最終顏色。

為獲得更平滑的結果，通常以像素中心為起點，向兩個相反方向（例如 -distance/2 到 +distance/2）進行採樣，這樣可以保持圖像內容的位置不變（中心模糊）。

## 參數說明

*   **角度 (Angle)**：運動的方向，以度為單位（0-360°）。
*   **距離 (Distance)**：運動的幅度，即模糊的長度（像素）。值越大，模糊程度越高，模擬的速度越快。

## 效能

由於此為空間域卷積（Spatial Convolution），計算複雜度與模糊距離成正比。對於大距離的模糊，計算量會顯著增加。使用 Web Worker 可以確保 UI 在計算過程中保持響應。
