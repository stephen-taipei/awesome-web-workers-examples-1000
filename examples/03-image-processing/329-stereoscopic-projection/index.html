About Stereoscopic Projection

Generate a Red-Cyan Anaglyph 3D image from a single 2D image by simulating binocular parallax. This creates a perception of depth when viewed with 3D glasses.

Anaglyph Method (Dubois):

1. Simulate Left/Right views by shifting the image horizontally.
   Left Image = Original shifted Left by Parallax/2
   Right Image = Original shifted Right by Parallax/2

2. Combine channels:
   R_out = Left_Red
   G_out = Right_Green
   B_out = Right_Blue
