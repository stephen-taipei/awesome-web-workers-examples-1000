About Gamma Correction

Gamma correction adjusts the luminance of an image using a power-law expression. It's essential for proper display calibration and can enhance details in shadows or highlights.

Gamma Correction Formula:

Power Law (Gamma) Transformation:

  Output = Input^γ

For normalized values (0 to 1):
  V_out = V_in^γ

For 8-bit values (0 to 255):
  V_out = 255 × (V_in / 255)^γ

Gamma Values:
  γ < 1.0  →  Brightens image (expands shadows)
  γ = 1.0  →  No change (linear)
  γ > 1.0  →  Darkens image (expands highlights)

Common Uses:
  γ = 0.45 (1/2.2)  →  sRGB encoding
  γ = 2.2          →  sRGB decoding
  γ = 0.5          →  Square root (brighten)
  γ = 2.0          →  Square (darken)

Inverse Gamma:
  To undo gamma correction: V_original = V_corrected^(1/γ)

Image Input

0.1 (Bright) 1.0 (Linear) 3.0 (Dark)
