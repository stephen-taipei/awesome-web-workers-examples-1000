Web Worker Example #269 - Blue Color Cast Effect

About Cool Tone Filter

The cool tone filter applies a blue/cyan color cast to images, creating a cold, winter-like atmosphere. This effect is achieved by enhancing blue channel values while reducing red tones, commonly used in landscape and portrait photography for moody aesthetics.

Cool Tone Transform:

For each pixel (R, G, B):

// Reduce red, slightly increase green, boost blue
newR = R × (1 - intensity × 0.2)
newG = G × (1 + intensity × 0.05)
newB = B × (1 + intensity × 0.3) + intensity × 20

Alternative Matrix Form (with saturation adjustment):
┌───────┐   ┌──────────────────────────────┐   ┌───┐
│ newR  │   │ 0.85   0.05   0.05   0  │   │ R │
│ newG  │ = │ 0.05   1.05   0.05   0  │ × │ G │
│ newB  │   │ 0.10   0.10   1.30   20 │   │ B │
│   1   │   │  0      0      0     1  │   │ 1 │
└───────┘   └──────────────────────────────┘   └───┘

Values are clamped to [0, 255]
