關於此範例

  • 濾鏡鏈：將多個濾鏡依序組合，形成處理流水線
  • Worker 類型：Dedicated Worker (專屬 Worker)
  • 通訊模式：使用 Transferable Objects 傳遞 ImageData
  • 支援濾鏡：灰階、反色、亮度、對比度、飽和度、懷舊、冷暖色調等
  • 效能特點：
    • 在背景執行緒處理，不阻塞主執行緒
    • 支援即時進度回報
    • 多濾鏡一次處理，減少資料傳輸開銷
