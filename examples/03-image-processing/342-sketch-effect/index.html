About Sketch Effect

Simulate a pencil sketch by blending an inverted, blurred version of the image with the original grayscale version using Color Dodge mode.

Processing Pipeline:

  1. Convert to Grayscale.
  2. Invert the grayscale image.
  3. Apply Gaussian Blur to the inverted image.
  4. Blend using Color Dodge: Final = min(Base / (1 - Blend), 255)
