# Canny 邊緣檢測 (Canny Edge Detection)

這個範例展示了如何使用 Web Workers 實作 Canny 邊緣檢測算法。

## 功能簡介

Canny 邊緣檢測是一種多階段算法，旨在檢測各種邊緣，具有低錯誤率、良好的定位和單一邊緣響應等特點。

## 技術重點與步驟

1.  **高斯濾波 (Gaussian Blur)**：使用高斯核平滑圖像，以減少噪聲對邊緣檢測的影響。`sigma` 參數控制平滑程度。
2.  **梯度計算 (Gradient Calculation)**：使用 Sobel 算子計算每個像素的梯度強度和方向。
3.  **非極大值抑制 (Non-Maximum Suppression)**：沿著梯度方向（量化為 0, 45, 90, 135 度），檢查當前像素是否為局部最大值。如果不是，則將其抑制為 0。這一步驟可以使邊緣變細（單像素寬）。
4.  **雙閾值 (Double Thresholding)**：
    *   **強邊緣**：梯度強度 > 高閾值。
    *   **弱邊緣**：低閾值 <= 梯度強度 <= 高閾值。
    *   **非邊緣**：梯度強度 < 低閾值。
5.  **邊緣追蹤 (Hysteresis Tracking)**：保留所有強邊緣。對於弱邊緣，只有當它們與強邊緣相連（8 鄰域連接）時才保留，否則捨棄。這確保了邊緣的連續性。

## 使用方法

1.  選擇或拖放一張圖片。
2.  調整 **Sigma** 參數以控制去噪程度。
3.  調整 **高閾值 (High)** 和 **低閾值 (Low)**。通常建議高閾值約為低閾值的 2 到 3 倍。
4.  點擊「偵測邊緣」查看結果。
