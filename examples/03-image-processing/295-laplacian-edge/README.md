# Laplacian 邊緣檢測 (Laplacian Edge Detection)

這個範例展示了如何使用 Web Workers 實作 Laplacian 邊緣檢測算子。

## 功能簡介

Laplacian 算子是一種二階微分算子，用於檢測影像中的快速變化區域（邊緣）。與 Sobel 和 Prewitt 這些一階算子不同，Laplacian 是各向同性的（Isotropic），這意味著它對所有方向的邊緣都同樣敏感，且不需要分別計算水平和垂直分量。

我們提供了兩種卷積核：
1.  **基礎核心 (4-neighbor)**：只考慮上下左右四個鄰居。
2.  **含對角核心 (8-neighbor)**：考慮周圍全部八個鄰居，對斜向邊緣更敏感。

## 技術重點

1.  **卷積運算**：使用 3x3 卷積核對圖像進行掃描。
    *   Basic:
        ```
         0  1  0
         1 -4  1
         0  1  0
        ```
    *   Diagonal:
        ```
         1  1  1
         1 -8  1
         1  1  1
        ```
2.  **絕對值處理**：二階導數在邊緣處會有正負值的變化（過零點），為了可視化邊緣強度，我們通常取計算結果的絕對值。

## 使用方法

1.  選擇或拖放一張圖片。
2.  選擇核心類型（基礎或含對角）。
3.  選擇顯示模式（灰階或二值化）。
4.  點擊「偵測邊緣」查看結果。
