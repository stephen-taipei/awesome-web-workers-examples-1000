About Fisheye Effect

Simulate a fisheye lens, which creates strong visual distortion intended to create a wide panoramic or hemispherical image. The barrel distortion maps the image onto a sphere-like surface.

Fisheye Formula:

Map (x, y) to normalized polar coordinates (r, theta).
r_src = atan(r * strength) / atan(strength)

Or simpler barrel approximation:
r_src = r * (1 + k * r^2)
