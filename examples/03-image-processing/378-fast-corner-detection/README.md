# FAST Corner Detection (快速角點檢測)

Web Worker Example #378

## 簡介

本範例展示如何使用 Web Worker 實現 FAST (Features from Accelerated Segment Test) 角點檢測算法。FAST 是一種運算速度極快的特徵點檢測算法，特別適用於即時影像處理應用。

## 功能

*   **高速檢測**: 使用 FAST-9 算法，檢查以像素為中心的圓周上是否有連續 9 個像素點亮度顯著不同。
*   **非極大值抑制 (NMS)**: 比較相鄰角點的得分，只保留局部響應最強的點，避免角點叢集。
*   **閾值調整**: 支援動態調整亮度差異閾值。

## 技術重點

1.  **快速測試**: 先檢查圓周上的 4 個點 (上下左右) 以快速排除非角點。
2.  **圓周檢測**: 檢查半徑為 3 的圓周上的 16 個像素。
3.  **位元操作/查表**: 雖然 JS 中無法像 C++ 那樣極致優化，但仍可通過減少記憶體存取來加速。

## 使用方式

1.  點擊上傳區域選擇圖片。
2.  調整 **Threshold** (閾值)。閾值定義了中心像素與圓周像素被認為是"不同"的最小亮度差。
3.  勾選 **Non-Maximal Suppression** 以獲得更精確的角點定位。
4.  點擊 "Detect Corners" 開始檢測。
