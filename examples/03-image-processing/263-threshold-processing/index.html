關於此範例

  • 閾值處理：將灰階影像轉換為二值影像，像素值高於閾值設為白色，低於則設為黑色
  • 全域閾值：使用單一固定閾值處理整張圖片
  • Otsu 演算法：自動計算最佳閾值，適合雙峰直方圖的影像
  • 自適應閾值：根據像素鄰域計算局部閾值，適合光照不均的影像
  • Worker 類型：Dedicated Worker (專屬 Worker)
  • 效能目標：1080p 圖片處理時間 < 50ms
