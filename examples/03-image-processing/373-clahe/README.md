# CLAHE (Contrast Limited Adaptive Histogram Equalization)

Web Worker Example #373

## 簡介

本範例展示如何使用 Web Worker 實現 CLAHE (限制對比度自適應直方圖均衡化)。CLAHE 是一種進階的圖像增強技術，它通過在圖像的小區域（Tile）上進行直方圖均衡化，並限制對比度增強的幅度，從而避免了普通直方圖均衡化可能帶來的噪聲放大問題。

## 功能

*   **自適應均衡化**: 將圖像分割成網格，對每個網格進行獨立的直方圖均衡化。
*   **對比度限制**: 限制直方圖的峰值，防止過度增強對比度導致的噪聲放大。
*   **雙線性插值**: 在重建圖像時使用雙線性插值平滑網格邊界，消除區塊效應。
*   **Lab 色彩空間**: 在 Lab 色彩空間的 L 通道上進行處理，保持色彩資訊。

## 技術重點

1.  **Lab 色彩空間轉換**: RGB <-> Lab 轉換算法。
2.  **分塊處理**: 將圖像分割為 NxM 個區塊。
3.  **直方圖裁剪**: 實現 Contrast Limit 邏輯。
4.  **雙線性插值**: 用於平滑區塊間的過渡。

## 使用方式

1.  點擊上傳區域選擇圖片。
2.  調整 **Tile Size** (網格大小) 和 **Clip Limit** (裁剪限制)。
    *   **Tile Size**: 決定局部處理的範圍大小。
    *   **Clip Limit**: 控制對比度增強的最大幅度，數值越大對比度越強，但噪聲也越多。
3.  點擊 "Apply CLAHE" 開始處理。
4.  下載結果圖片。
