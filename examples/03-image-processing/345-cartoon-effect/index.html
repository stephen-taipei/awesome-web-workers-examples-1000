About Cartoon Effect

Create a cartoon or "cel-shaded" look by reducing the color palette (quantization) and adding bold outlines to edges.

Processing Pipeline:

  1. Color Quantization: Reduce colors to a few discrete levels (e.g., floor(color / 32) * 32).
  2. Smoothing: Apply a median or bilateral filter to flatten colors.
  3. Edge Detection: Detect edges and overlay them as black lines.
