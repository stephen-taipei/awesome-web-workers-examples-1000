橢圓檢測 (Ellipse Detection)

利用 Web Worker 與最小平方法擬合檢測橢圓

Web Worker Least Squares Fitting Expert

功能描述

本示例演示如何檢測圖片中的橢圓。橢圓檢測比圓形檢測更複雜，因為它有 5 個自由度（中心 x, y，長軸 a，短軸 b，旋轉角度 θ）。本示例首先提取邊緣並尋找連通區域，然後對每個可能的邊緣段進行最小二乘橢圓擬合。

Ax² + Bxy + Cy² + Dx + Ey + F = 0 Fitting: Minimize Σ(Axᵢ² + Bxᵢyᵢ + Cyᵢ² + Dxᵢ + Eyᵢ + F)² Constraint: B² - 4AC < 0
