# 橢圓檢測 (Ellipse Detection)

本示例展示了如何使用 Web Workers 和基於矩 (Moments) 的擬合算法來檢測圖像中的橢圓。

## 技術重點

1.  **邊緣提取與輪廓追蹤**:
    - 計算圖像梯度並提取邊緣。
    - 使用連通區域算法 (Connected Components) 提取獨立的邊緣輪廓。

2.  **橢圓擬合 (Ellipse Fitting)**:
    - 對於每個提取的輪廓，計算其二階中心矩 (Second Order Central Moments)。
    - 通過協方差矩陣的特徵值分解，估算橢圓的長軸、短軸和旋轉角度。
    - 這種方法對於閉合或接近閉合的形狀效果較好。

## 使用說明

1.  上傳包含橢圓物體的圖片（如雞蛋、行星、圓形物體的透視圖）。
2.  點擊「開始檢測」。
3.  系統將在右側標示出檢測到的橢圓（粉紅色）。

## 注意事項

- 本示例使用的是基於矩的簡化擬合方法，對於不完整的橢圓弧段，擬合效果可能不如最小二乘法 (Least Squares) 精確。
- 為了獲得最佳效果，圖像背景應較為乾淨，橢圓邊緣應清晰。
- 複雜場景下可能需要先進行圖像分割或更高級的邊緣連接處理。
