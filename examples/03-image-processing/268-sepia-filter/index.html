About Sepia Filter

The sepia tone filter applies a warm, brownish tint to images, creating a vintage or antique appearance. This effect mimics the look of old photographs from the late 19th and early 20th centuries.

Sepia Color Matrix Transform:

For each pixel (R, G, B):

newR = 0.393*R + 0.769*G + 0.189*B
newG = 0.349*R + 0.686*G + 0.168*B
newB = 0.272*R + 0.534*G + 0.131*B

Matrix Form:
┌───────┐   ┌─────────────────────────┐   ┌───┐
│ newR  │   │ 0.393  0.769  0.189  0 │   │ R │
│ newG  │ = │ 0.349  0.686  0.168  0 │ × │ G │
│ newB  │   │ 0.272  0.534  0.131  0 │   │ B │
│   1   │   │   0      0      0    1 │   │ 1 │
└───────┘   └─────────────────────────┘   └───┘

Values are clamped to [0, 255]
