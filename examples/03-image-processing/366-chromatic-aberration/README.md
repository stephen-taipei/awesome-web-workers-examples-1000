# Chromatic Aberration

此示例展示了如何使用 Web Worker 模擬色差（Chromatic Aberration）效果。

## 技術原理

色差是由於透鏡無法將所有顏色的光聚焦在同一點上而產生的光學現象。在數位圖像處理中，這通常通過分離 RGB 通道並對其進行位移來模擬。

此示例提供了兩種模式：
1.  **徑向 (Radial)**：模擬真實鏡頭邊緣的色差。紅光和藍光沿著從圖像中心向外的方向發生相對位移。位移量通常與距離中心的距離成正比（或平方比）。
2.  **線性 (Linear)**：將紅光和藍光沿著特定方向進行均勻位移。這種效果較不自然，但常用於藝術風格化（如故障藝術）。

## 參數說明

*   **類型 (Type)**：選擇徑向或線性色差。
*   **強度 (Intensity)**：控制 RGB 通道分離的距離。
*   **方向 (Direction)**：僅在線性模式下有效，控制位移的角度。

## 演算法細節

Worker 遍歷圖像的每個像素，分別計算 R、G、B 三個通道的採樣座標。
*   **G 通道**：保持不變（作為基準）。
*   **R/B 通道**：根據選擇的模式和強度，計算偏移後的座標並進行採樣（Nearest Neighbor 插值）。

```javascript
// 線性位移範例
rX = x - dirX * intensity;
rY = y - dirY * intensity;
RedValue = Sample(rX, rY);
```
