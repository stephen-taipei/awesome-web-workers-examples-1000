# 距離變換 (Distance Transform)

距離變換是一種圖像處理操作，它計算每個像素到最近的背景像素（零像素）的距離。這通常用於二值圖像，其結果是一個灰階圖像，其中像素的亮度代表其距離。

## 原理

本示例使用 **歐幾里得距離變換 (Euclidean Distance Transform)**。
為了高效計算，我們使用了 **Meijster 算法**，這是一種線性時間複雜度 O(N) 的算法，將 2D 距離變換分解為兩個獨立的 1D 掃描：

1.  **列掃描 (Column scan)**：計算每列中每個像素到最近背景像素的垂直距離（平方）。
2.  **行掃描 (Row scan)**：利用列掃描的結果，計算每行的下包絡線（Lower Envelope of Parabolas），從而得到最終的 2D 歐幾里得距離。

## 使用方式

1.  上傳圖片。
2.  調整「二值化閾值」：定義哪些是前景（物體），哪些是背景。距離是從前景內部計算到最近背景邊界的距離。
3.  點擊「計算距離變換」。

## 應用場景

*   **骨架提取**：距離變換的脊線即為物體的骨架。
*   **分水嶺算法**：作為分水嶺分割的標記圖像。
*   **形狀匹配**：Hausdorff 距離計算。
*   **模糊效果**：可變半徑的模糊。

## 技術實現

*   **Web Worker**：雖然算法是 O(N)，但對於大圖像（如 4K），計算仍然密集，因此在 Worker 中執行。
*   **Float32Array**：用於存儲中間計算結果（平方距離），避免精度損失或溢出。
