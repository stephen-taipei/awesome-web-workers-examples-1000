黑帽變換 (Black-hat)

Image Morphological Black-hat Transform

Web Worker Image Processing Morphology

功能描述

黑帽變換是形態學閉運算 (Closing) 與原圖的差。它用於提取比周圍環境暗的圖像區域，或者用於校正不均勻的背景。

BlackHat(f) = Closing(f) - f Closing(f) = Erosion(Dilation(f))

參數設定

值越大，提取的區域越大，運算越慢。
