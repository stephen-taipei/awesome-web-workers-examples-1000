About Oil Painting Effect

Simulate an oil painting using the Kuwahara filter. This non-linear smoothing filter reduces noise while preserving edges, creating a blocky, brush-stroke-like appearance.

Kuwahara Filter Algorithm:

  1. Divide a window around each pixel into 4 quadrants.
  2. Calculate the mean and variance of brightness for each quadrant.
  3. Choose the quadrant with the lowest variance (most uniform).
  4. Set the central pixel to the mean color of that chosen quadrant.
