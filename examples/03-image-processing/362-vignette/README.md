# 暈影效果 (Vignette)

本範例展示如何使用 Web Worker 為圖像添加暈影效果（Vignette Effect），使圖像邊緣逐漸變暗，從而突出中心主體。

## 功能簡介

- **暈影生成**：根據像素到中心的距離，計算衰減係數。
- **參數調整**：
    -   **強度 (Strength)**：控制暈影的深淺。
    -   **半徑 (Radius)**：控制暈影開始出現的範圍（中心清晰區域的大小）。
    -   **顏色 (Color)**：可自定義暈影的顏色（默認為黑色）。

## 技術重點

1.  **徑向漸變 (Radial Gradient)**：
    -   計算每個像素相對於圖像中心的歐氏距離。
    -   使用平滑函數（如 Smoothstep）將距離映射到混合因子。
    -   根據混合因子將原始像素顏色與暈影顏色進行混合。
    -   $ Color_{out} = Color_{in} \times (1 - factor) + Color_{vignette} \times factor $

2.  **Web Worker**：
    -   雖然簡單的暈影可以用 CSS `box-shadow` 或 `radial-gradient` 實現，但在 Canvas 中進行像素級處理可以將效果永久固化在圖像數據中，方便後續保存或進一步處理。

## 使用方式

1.  開啟網頁。
2.  拖放或選擇一張圖片上傳。
3.  調整「強度」和「半徑」滑桿。
4.  選擇「暈影顏色」。
5.  點擊「應用效果」按鈕。
