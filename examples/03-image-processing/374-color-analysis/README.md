# Color Analysis (顏色分析)

Web Worker Example #374

## 簡介

本範例展示如何使用 Web Worker 進行圖片的主要顏色分析。透過 K-Means 聚類演算法，我們可以從圖片中提取出最具代表性的顏色（主色調），並分析各個顏色的佔比。

## 功能

*   **K-Means 聚類**: 自動將圖片中的像素顏色分組為 K 個聚類。
*   **主色提取**: 找出每個聚類的中心點作為代表色。
*   **佔比分析**: 計算每個主色在圖片中的分布比例。
*   **視覺化**: 顯示色票與統計圖表。

## 技術重點

1.  **K-Means 算法**: 在 RGB 顏色空間中運行的非監督式學習算法。
2.  **迭代優化**: 通過多次迭代更新聚類中心，直到收斂。
3.  **效能優化**: 在 Web Worker 中處理大量像素數據的距離計算。

## 使用方式

1.  點擊上傳區域選擇圖片。
2.  調整 **Number of Colors (K)** 滑桿，設定要提取的主色數量。
3.  點擊 "Analyze Colors" 開始分析。
4.  查看提取出的色票與顏色分布圖表。
