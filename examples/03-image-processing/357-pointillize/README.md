# 斑點化效果 (Pointillize)

本範例展示如何使用 Web Worker 實現圖像的斑點化效果，模擬點描畫派（Pointillism）的藝術風格。

## 功能簡介

- **斑點生成**：在畫布上隨機生成大量圓形斑點。
- **色彩採樣**：每個斑點的顏色採樣自原圖對應位置的顏色。
- **密度控制**：可調整斑點的大小與生成密度。

## 技術重點

1.  **隨機採樣 (Random Sampling)**：
    -   隨機選擇圖像上的位置作為斑點中心。
    -   讀取該位置的像素顏色作為斑點顏色。

2.  **像素級繪圖 (Pixel Manipulation)**：
    -   在 Web Worker 中直接操作 `Uint8ClampedArray` 像素數據。
    -   手動實現圓形繪製算法（檢查像素與中心的距離）。
    -   雖然使用 `OffscreenCanvas` 可能更方便，但直接操作像素陣列能確保在所有支援 Web Worker 的環境中運作，且能深入理解圖像處理底層。

3.  **效能考量**：
    -   大量斑點（數萬至數十萬）的繪製計算量大，適合移至 Worker 執行。
    -   通過簡單的邊界框檢查優化圓形繪製。

## 使用方式

1.  開啟網頁。
2.  拖放或選擇一張圖片上傳。
3.  調整「斑點大小」滑桿，控制每個點的半徑。
4.  選擇「密度」等級，控制生成的點數多寡。
5.  點擊「應用效果」按鈕開始處理。
