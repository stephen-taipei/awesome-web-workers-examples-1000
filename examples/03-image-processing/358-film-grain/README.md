# 顆粒效果 (Film Grain)

本範例展示如何使用 Web Worker 為圖像添加顆粒效果（Film Grain），模擬傳統底片的質感。

## 功能簡介

- **噪聲生成**：生成隨機噪聲並疊加到圖像上。
- **強度調整**：可調整噪聲的強度（幅度）。
- **彩色/單色噪聲**：支持添加彩色噪聲或灰度噪聲。

## 技術重點

1.  **噪聲疊加**：
    -   對於每個像素，生成一個隨機值（噪聲）。
    -   將噪聲值加到原始像素的 RGB 分量上。
    -   彩色噪聲：R、G、B 通道使用獨立的隨機值。
    -   單色噪聲：R、G、B 通道使用相同的隨機值。

2.  **效能考量**：
    -   像素級遍歷操作適合在 Web Worker 中並行處理。
    -   使用 `Uint8ClampedArray` 自動處理數值溢出（0-255 範圍限制）。

## 使用方式

1.  開啟網頁。
2.  拖放或選擇一張圖片上傳。
3.  調整「顆粒強度」滑桿。
4.  勾選或取消「彩色噪聲」選項。
5.  點擊「應用效果」按鈕查看結果。
