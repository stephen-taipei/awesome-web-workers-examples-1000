# 浮雕效果 (Emboss Effect)

這個範例展示了如何使用 Web Workers 實作浮雕濾鏡效果。

## 功能簡介

浮雕濾鏡通過計算像素在特定方向上的亮度差分，模擬出光照照射在凹凸表面上的效果。
平坦區域（顏色變化小）會變成灰色（因為差分為 0，加上偏移量 128 後變為中性灰）。
邊緣區域（顏色變化大）會根據變化方向呈現亮色（受光面）或暗色（背光面），從而產生立體感。

## 技術重點

1.  **卷積運算**：使用 3x3 的卷積核。核心的權重和為 0，這意味著對於均勻顏色的區域，卷積結果為 0。
2.  **方向性**：不同的卷積核模擬不同的光源方向。
    *   例如左上光源 (Top-Left)：
        ```
        -s -s  0
        -s  0  s
         0  s  s
        ```
        (其中 s 為強度係數)
3.  **灰階偏移**：將卷積結果加上 128，使結果落在可視範圍內，並以中性灰作為基準。
4.  **灰階處理**：浮雕效果通常在灰階模式下表現最好，因此我們提供了一個選項將結果轉換為灰階。

## 使用方法

1.  選擇或拖放一張圖片。
2.  選擇 **光源方向**（如左上、上方、右方等）。
3.  調整 **強度**。強度越高，凹凸感越強烈，對比度越高。
4.  選擇是否 **轉為灰階**（預設勾選）。
5.  點擊「套用濾鏡」查看結果。
