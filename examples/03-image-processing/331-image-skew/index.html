About Image Skew

Apply a shear (skew) transformation to the image. This slants the image horizontally or vertically, simulating perspective or motion.

Shear Matrix:

| x' |   | 1  shx | | x |
|    | = |        | |   |
| y' |   | shy  1 | | y |

x' = x + shx * y
y' = y + shy * x
