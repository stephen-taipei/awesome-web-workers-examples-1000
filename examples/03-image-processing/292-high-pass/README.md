# 高反差保留 (High Pass)

這個範例展示了如何使用 Web Workers 實作高反差保留濾鏡。

## 功能簡介

高反差保留濾鏡（High Pass Filter）是一種影像處理技術，用於保留影像中的高頻細節（如邊緣、紋理），同時抑制低頻成分（如平滑的顏色漸變）。

在數位影像處理中，通常的實作方式為：
1. 對原始影像應用模糊濾鏡（如高斯模糊）。
2. 將原始影像減去模糊後的影像。
3. 加上 128（中性灰）作為偏移量，使結果可視化。

## 技術重點

1.  **卷積運算 (Convolution)**：雖然這裡使用了「原圖 - 模糊圖」的方法，但模糊本身就是一種卷積。
2.  **像素操作**：直接操作 `ImageData` 的 `Uint8ClampedArray`。
3.  **分離式濾波**：為了效能，模糊過程採用了分離式（Separable）運算，先進行水平模糊，再進行垂直模糊，將時間複雜度從 O(R²) 降低到 O(R)。

## 使用方法

1.  選擇或拖放一張圖片。
2.  調整「半徑」參數。半徑越小，保留的細節越細微（主要剩邊緣）；半徑越大，保留的層次越多（看起來像褪色的原圖）。
3.  點擊「套用濾鏡」查看結果。
