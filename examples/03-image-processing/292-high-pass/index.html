關於高反差保留 (High Pass)

  • 原理：從原圖中減去低頻成分（模糊後的影像），保留高頻成分（邊緣和細節）。
  • 公式：HighPass = Original - Blurred + 128
  • 偏移量 128：為了讓結果顯示在 0-255 的範圍內，通常會加上 128（中性灰），使得平坦區域呈現灰色，邊緣呈現亮或暗。
  • 用途：
    • 圖像銳化 (Sharpening)：將高反差結果疊加到原圖上（如 "Overlay" 混合模式）。
    • 邊緣檢測：檢視圖片的細節和噪點。
