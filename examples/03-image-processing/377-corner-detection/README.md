# Harris Corner Detection (Harris 角點檢測)

Web Worker Example #377

## 簡介

本範例展示如何使用 Web Worker 實現 Harris 角點檢測算法。角點是圖像中重要的特徵點，Harris 算子透過計算圖像梯度在各個方向上的變化量來識別這些點。

## 功能

*   **梯度計算**: 使用 Sobel 算子計算水平和垂直方向的梯度。
*   **結構張量**: 計算並平滑結構張量矩陣 (M)。
*   **Harris 響應**: 計算角點響應函數 R = det(M) - k(trace(M))^2。
*   **非極大值抑制**: 在局部區域內尋找響應值最大的點作為最終角點。

## 技術重點

1.  **影像處理管道**: 灰階化 -> 梯度計算 -> 高斯平滑 -> 響應計算 -> 局部極值篩選。
2.  **卷積運算**: 涉及多次 3x3 或 5x5 的卷積核運算。
3.  **數學運算**: 矩陣行列式與跡的計算。

## 使用方式

1.  點擊上傳區域選擇圖片。
2.  調整 **Threshold** (閾值)。數值越高，檢測到的角點越少但越顯著；數值越低，檢測到的角點越多但可能包含噪聲。
3.  點擊 "Detect Corners" 開始檢測。
4.  結果圖像會標記出紅色的角點。
