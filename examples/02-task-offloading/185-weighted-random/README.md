# 加權隨機負載均衡 (Weighted Random)

此示例演示如何使用加權隨機算法 (Weighted Random) 在多個 Web Worker 之間分配任務。

## 功能

1.  **加權隨機分配**：為每個 Worker 配置權重，權重越高，被選中的機率越大。
2.  **機率視覺化**：直觀顯示每個 Worker 的理論選中機率和實際分配佔比。
3.  **統計驗證**：隨著任務數量增加，實際分配佔比應趨近於理論機率。

## 算法說明

假設有 $N$ 個 Worker，權重分別為 $W_1, W_2, ..., W_N$。
總權重 $Total = \sum W_i$。

Worker $i$ 被選中的機率為 $P_i = \frac{W_i}{Total}$。

實現方式：
1. 計算總權重 $Total$。
2. 產生一個 $[0, Total)$ 區間內的隨機數 $R$。
3. 遍歷 Worker，從 $R$ 中減去該 Worker 的權重 $W_i$。
4. 當 $R \le 0$ 時，選擇當前 Worker。

## 使用方式

1.  在 "Worker 配置" 區域調整每個 Worker 的權重，點擊 "更新權重"。
2.  觀察下方 Worker 卡片中 "選擇機率" 的變化。
3.  開啟 "自動產生任務"，觀察大量任務下的 "實際佔比"。
4.  對比 "選擇機率" 與 "實際佔比"，驗證加權隨機算法的效果。
