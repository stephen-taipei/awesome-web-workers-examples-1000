# Worker 池監控儀表板

這個範例展示了如何建立一個 Web Worker 池並使用儀表板即時監控其狀態。

## 功能特點

1.  **Worker 池管理**：動態新增或移除 Worker。
2.  **任務排程**：將任務放入佇列，並由空閒的 Worker 處理。
3.  **即時監控**：
    *   使用 Canvas 繪製即時圖表，顯示佇列長度與活躍 Worker 數量。
    *   每個 Worker 的詳細狀態卡片（忙碌/空閒、進度條）。
    *   全域統計資訊（總任務、完成數、處理時間）。
4.  **互動控制**：
    *   手動提交任務。
    *   批量提交任務。
    *   自動隨機投遞任務以模擬負載。

## 技術重點

*   **Web Workers**: 用於背景執行模擬的繁重計算任務。
*   **Canvas API**: 用於繪製即時的系統負載圖表。
*   **佇列管理**: 簡單的 FIFO 佇列來管理待處理任務。
*   **狀態同步**: 主執行緒維護 Worker 池的完整狀態並即時更新 UI。

## 使用方式

1.  打開 `index.html`。
2.  點擊「新增 Worker」增加處理能力。
3.  點擊「提交隨機任務」或「開啟自動投遞任務」來產生負載。
4.  觀察儀表板上的圖表變化與 Worker 卡片的狀態更新。
