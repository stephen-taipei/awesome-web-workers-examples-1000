# Web Worker 限流器示例

此示例演示了如何使用 Web Worker 實現基於**令牌桶演算法 (Token Bucket)** 的請求限流器。

## 功能

1.  **請求發送**: 模擬發送請求。
2.  **突發請求**: 模擬短時間內的大量請求。
3.  **動態配置**: 可調整桶容量 (Capacity) 和填充速率 (Refill Rate)。
4.  **即時監控**: 顯示當前令牌數量和請求結果。

## 技術重點

*   **Web Worker**: 限流邏輯完全在 Worker 中執行，不阻塞主執行緒 UI。
*   **Token Bucket**: 實現標準的令牌桶演算法。
*   **非同步通訊**: 主執行緒發送請求意圖，Worker 返回准許/拒絕結果。

## 使用方式

1.  打開 `index.html`。
2.  點擊「發送請求」按鈕，觀察令牌數減少。
3.  點擊「發送突發請求」按鈕，觀察在令牌耗盡後的限流行為。
4.  調整「桶容量」和「填充速率」並點擊「更新設定」以改變限流行為。
