About Data Parallelism

Data Parallelism (SIMD - Single Instruction, Multiple Data) is a parallel computing paradigm where the same operation is applied to multiple data elements simultaneously. Data is split across workers, each performing identical operations on their portion, then results are combined.

Data Parallelism Model:

┌─────────────────────────────────────────────────┐
│              Original Data Array                │
│  [d₀, d₁, d₂, d₃, d₄, d₅, d₆, d₇, ... dₙ]     │
└─────────────────────────────────────────────────┘
                    ↓ Split
    ┌───────────┬───────────┬───────────┐
    ↓           ↓           ↓           ↓
┌───────┐  ┌───────┐  ┌───────┐  ┌───────┐
│Worker1│  │Worker2│  │Worker3│  │Worker4│
│[d₀-d₂]│  │[d₃-d₅]│  │[d₆-d₈]│  │[d₉-dₙ]│
│  f(x) │  │  f(x) │  │  f(x) │  │  f(x) │
└───────┘  └───────┘  └───────┘  └───────┘
    ↓           ↓           ↓           ↓
    └───────────┴───────────┴───────────┘
                    ↓ Combine
┌─────────────────────────────────────────────────┐
│               Result Array                       │
│  [r₀, r₁, r₂, r₃, r₄, r₅, r₆, r₇, ... rₙ]     │
└─────────────────────────────────────────────────┘

Common Operations:
• Map: Apply function to each element
• Filter: Select elements matching condition
• Reduce: Aggregate to single value
• Scan: Prefix sum / cumulative operation
