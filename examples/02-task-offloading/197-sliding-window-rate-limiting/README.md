# Web Worker 滑動窗口限流 (Sliding Window Rate Limiter)

此示例演示了如何使用 Web Worker 實現**滑動窗口 (Sliding Window)** 限流演算法。

## 功能

1.  **請求限流**: 在指定時間窗口內限制請求數量。
2.  **可視化窗口**: 動態顯示請求在時間窗口內的移動和過期過程。
3.  **動態配置**: 可即時調整窗口大小和限制數量。

## 技術重點

*   **滑動窗口演算法**: 記錄每個請求的時間戳，並在每次請求時清除過期的時間戳，從而精確控制任意時間窗口內的請求量。
*   **視覺化**: 使用 CSS 和 JavaScript 動畫來直觀展示「滑動窗口」的概念。
*   **Web Worker**: 負責維護請求隊列和判定邏輯。

## 使用方式

1.  設定「窗口大小」和「限制請求數」。
2.  點擊「發送請求」或「發送突發請求」。
3.  觀察下方的可視化區域：
    *   綠點代表被允許的請求。
    *   紅點代表被拒絕的請求。
    *   點會從右向左移動，一旦移出左側邊界，即表示該請求已「過期」，不再計入當前窗口的限制。
