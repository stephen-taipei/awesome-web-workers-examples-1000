# 綜合基準測試 (Comprehensive Benchmark)

這個範例展示了如何在 Web Worker 中建立一個綜合效能測試套件。它包含 CPU（整數/浮點運算）、記憶體（順序/隨機存取）以及腳本執行（JSON 解析）的測試。

## 測試項目

1.  **整數運算**: 大量的 XOR 與取模運算。
2.  **浮點運算**: 乘法與三角函數計算。
3.  **記憶體**: TypedArray 的順序寫入與隨機讀取。
4.  **JSON**: 複雜物件的序列化與反序列化。

## 評分機制

每個測試項目根據執行時間 (Operations Per Second) 計算分數，最後加總為總分。分數僅供參考，用於比較不同瀏覽器或設備的相對效能。
