# 加密效能測試 (Crypto Performance)

本範例展示使用 Web Workers 結合 Web Crypto API 進行高效的加密運算。

## 功能簡介

- **SHA-256 雜湊**：計算大數據塊的 SHA-256 摘要。
- **AES-GCM 加密**：生成 256 位元金鑰並對數據進行對稱加密。

## 技術重點

- **Web Crypto API**：現代瀏覽器提供的原生加密 API，通常由 C++ 層級實現，速度遠快於 JavaScript 實現的加密庫。
- **Worker 併發**：加密運算（尤其是大文件）是 CPU 密集型的，移至 Worker 可以確保主執行緒流暢。

## 使用方式

1.  選擇「數據大小」。
2.  點擊「測試雜湊」或「測試加密」。
3.  查看耗時及吞吐量。
