About Structured Clone

  • Supported: Primitives, Date, RegExp, Blob, File, Map, Set, ArrayBuffer
  • Unsupported: Functions, DOM nodes, Error objects (partially)
  • Behavior: Deep clone - no shared references
  • Performance: Copy overhead for large data
