關於此範例

  • 功能：確保非同步處理的訊息按照原始順序輸出，即使訊息以亂序到達
  • 序號機制：每則訊息都帶有唯一序號，用於追蹤和重新排序
  • 緩衝區：提前到達的訊息會暫存於緩衝區，等待前面的訊息處理完畢
  • Worker 類型：Dedicated Worker (專屬 Worker)
  • 應用場景：視訊串流、日誌處理、交易系統等需要保證順序的場景
