Web Worker Example #234 - 散佈-收集模式

About Scatter-Gather Pattern

Scatter-Gather 模式將資料分散（Scatter）到多個 Worker 並行處理，然後收集（Gather）所有結果進行彙整。這種模式適用於批量處理場景，如搜尋引擎查詢多個資料源、分散式計算等。

Scatter-Gather 執行流程：

                    ┌─────────────┐
                    │   Request   │
                    └──────┬──────┘
                           │ SCATTER (分散)
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
    ┌────────────┐  ┌────────────┐  ┌────────────┐
    │  Worker 1  │  │  Worker 2  │  │  Worker 3  │
    │  (Shard A) │  │  (Shard B) │  │  (Shard C) │
    └─────┬──────┘  └─────┬──────┘  └─────┬──────┘
          │               │               │
          ▼               ▼               ▼
    ┌────────────┐  ┌────────────┐  ┌────────────┐
    │  Result A  │  │  Result B  │  │  Result C  │
    └─────┬──────┘  └─────┬──────┘  └─────┬──────┘
          │               │               │
          └───────────────┼───────────────┘
                          │ GATHER (收集)
                          ▼
                 ┌─────────────────┐
                 │  Merged Result  │
                 └─────────────────┘

應用場景：
• 分散式搜尋 (Search across shards)
• 並行資料庫查詢
• Map-Reduce 的 Map 階段
• 服務聚合 (Service Aggregation)
