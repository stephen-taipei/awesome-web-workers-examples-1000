關於此範例

  • 功能：為訊息設定 TTL (Time To Live)，超過時間自動清理
  • TTL 機制：訊息建立時記錄時間戳，Worker 定期檢查並清理過期訊息
  • 清理策略：支援定時清理與立即清理兩種模式
  • Worker 類型：Dedicated Worker (專屬 Worker)
  • 視覺化：即時顯示訊息剩餘時間與狀態
  • 應用場景：快取管理、會話超時、暫存資料清理等
