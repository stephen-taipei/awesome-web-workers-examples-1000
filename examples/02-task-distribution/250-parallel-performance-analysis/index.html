About Parallel Performance Analysis

This example demonstrates how to measure and analyze parallel execution performance using Amdahl's Law and related metrics. Understanding these principles helps optimize worker configurations for real-world applications.

Amdahl's Law & Performance Metrics:

Amdahl's Law:
  Speedup(n) = 1 / ((1-P) + P/n)
  Where:
    P = Parallelizable fraction (0-1)
    n = Number of processors
    1-P = Serial fraction

Maximum Speedup:
  lim(n→∞) Speedup = 1/(1-P)

Gustafson's Law (Scaled Speedup):
  Speedup(n) = n - (1-P)(n-1)

Efficiency:
  E = Speedup / n = Throughput_parallel / (n × Throughput_serial)

Karp-Flatt Metric:
  e = (1/Speedup - 1/n) / (1 - 1/n)
  Measures serial fraction including overhead
