關於此範例

  • 功能：自動檢測並過濾重複訊息，防止同一訊息被多次處理
  • 內容雜湊：使用 SHA-256 計算訊息內容的雜湊值進行比對
  • 訊息 ID：透過唯一訊息 ID 進行快速去重檢測
  • LRU 快取：使用 LRU (最近最少使用) 策略管理雜湊快取
  • Worker 類型：Dedicated Worker (專屬 Worker)
  • 應用場景：訊息佇列、API 請求去重、事件處理等
