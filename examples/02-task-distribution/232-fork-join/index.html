About Fork-Join Pattern

Fork-Join 是一種分治並行模式，將大型任務遞迴分解為較小的子任務（Fork），並行處理後再合併結果（Join）。這種模式特別適用於可以自然分割的問題，如陣列處理、排序、搜尋等。

Fork-Join 執行流程：

┌─────────────────────────────────────────────┐
│                Main Task                     │
└─────────────────┬───────────────────────────┘
                  │ FORK (分割)
        ┌─────────┼─────────┐
        ▼         ▼         ▼
    ┌───────┐ ┌───────┐ ┌───────┐
    │Task 1 │ │Task 2 │ │Task 3 │
    └───┬───┘ └───┬───┘ └───┬───┘
        │         │         │
        ▼         ▼         ▼
    ┌───────┐ ┌───────┐ ┌───────┐
    │Result1│ │Result2│ │Result3│
    └───────┘ └───────┘ └───────┘
        │         │         │
        └─────────┼─────────┘
                  │ JOIN (合併)
                  ▼
         ┌────────────────┐
         │  Final Result  │
         └────────────────┘

應用場景：
• 大型陣列的平行處理
• Merge Sort / Quick Sort
• 矩陣運算
• 檔案批次處理
