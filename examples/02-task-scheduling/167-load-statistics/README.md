# 負載統計 (Load Statistics)

此示例展示如何收集與視覺化 Worker 池的負載數據。

## 功能說明

1.  **任務計數**: 統計總任務數、已完成任務數。
2.  **效能監控**: 計算平均任務執行時間。
3.  **Worker 狀態監控**:
    *   **個別統計**: 追蹤每個 Worker 完成的任務數量和總工作時間。
    *   **狀態顯示**: 即時顯示 Worker 是閒置還是忙碌。
4.  **視覺化圖表**: 使用 CSS 繪製簡易長條圖，比較不同 Worker 的負載（已完成任務數）。
5.  **負載生成**: 提供按鈕生成大量隨機任務，以觀察統計數據的變化。

## 使用方式

1.  開啟 `index.html`。
2.  點擊「新增小型任務」或「新增大型任務」來觀察單一任務對統計數據的影響。
3.  點擊「隨機生成 50 個任務」來模擬高負載場景。
4.  觀察儀表板上的數字變化、圖表的長條高度變化，以及下方每個 Worker 卡片的詳細數據。
5.  日誌面板會顯示每個任務的完成時間與負責的 Worker。

## 技術重點

*   **Metric Collection**: 在任務分發與完成的生命週期中收集數據。
*   **Data Aggregation**: 聚合原始數據計算平均值與總和。
*   **Real-time Visualization**: 即時將數據反映到 UI 和圖表上。
