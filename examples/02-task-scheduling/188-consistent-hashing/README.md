# 一致性雜湊 (Consistent Hashing)

此示例演示了一致性雜湊演算法的原理和視覺化效果。

## 功能

1.  **節點管理**：可以新增或移除節點 (伺服器)。
2.  **鍵管理**：可以新增鍵 (任務或數據) 到系統中。
3.  **虛擬節點**：支援配置每個真實節點對應的虛擬節點數量，觀察其對負載均衡的影響。
4.  **視覺化**：
    *   **雜湊環**：顯示一個環代表 0 到 2^32 的雜湊空間。
    *   **虛擬節點**：在環上標記虛擬節點的位置，顏色對應真實節點。
    *   **鍵分配**：顯示鍵在環上的位置，並用虛線連接到負責該鍵的虛擬節點。
5.  **統計**：即時顯示每個節點分配到的鍵數量和百分比。

## 技術重點

*   **Web Worker**：所有的雜湊計算、環的維護和查找邏輯都在 Worker 中運行，避免阻塞主執行緒。
*   **Canvas API**：使用 Canvas 繪製雜湊環和節點分布。
*   **Hashing**：使用 FNV-1a 簡單雜湊演算法將字串映射到 32 位整數。

## 使用方式

1.  開啟 `index.html`。
2.  使用控制面板新增節點。
3.  新增一些鍵 (Key)，觀察它們如何分配到節點。
4.  嘗試調整虛擬節點數量，觀察分布的均勻程度變化。
5.  移除一個節點，觀察只有部分鍵被重新分配 (一致性雜湊的特性)。
