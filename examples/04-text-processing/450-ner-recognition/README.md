# 命名實體識別 (NER)

本範例展示如何使用 Web Worker 進行基於規則的命名實體識別 (Named Entity Recognition)。

## 功能簡介

- **實體識別**：
    -   **電子郵件 (EMAIL)**：識別符合標準格式的電子郵件地址。
    -   **日期 (DATE)**：識別 YYYY-MM-DD 等常見日期格式。
    -   **電話 (PHONE)**：識別常見的電話號碼格式。
    -   **人名/地名 (PERSON/LOCATION)**：基於大寫字母和關鍵詞的簡單啟發式識別。

## 技術重點

1.  **正則表達式 (Regex)**：
    -   使用 JS 的 `RegExp` 對文本進行全域掃描。
    -   利用 `exec` 方法的循環來捕獲所有匹配項及其位置。

2.  **重疊處理**：
    -   多個規則可能會匹配到重疊的區域。
    -   本範例實現了簡單的衝突解決邏輯，按位置排序並丟棄重疊的後續匹配。

3.  **HTML 注入防護**：
    -   在生成帶有高亮標籤的 HTML 時，對原始文本進行轉義，防止 XSS 攻擊。

## 使用方式

1.  在文本框中輸入或粘貼文本。
2.  點擊「開始識別」按鈕。
3.  結果中將高亮顯示識別出的實體，鼠標懸停可查看實體類型。
