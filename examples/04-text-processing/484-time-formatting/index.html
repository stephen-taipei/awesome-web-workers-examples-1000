How it works

  • The main thread sends timestamps or durations to a Web Worker.
  • The Worker calculates relative time using `Intl.RelativeTimeFormat` (e.g., "5 minutes ago", "in 2 hours").
  • Supports formatting durations (HH:MM:SS) and standard clock times.
  • Efficiently handles massive lists of timestamps.
