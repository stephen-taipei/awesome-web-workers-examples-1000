# Fuzzy Search (模糊搜尋)

Web Worker Example #433

## 簡介

本範例展示如何使用 Web Worker 進行模糊搜尋。模糊搜尋允許使用者輸入不完全準確的關鍵字，系統會根據編輯距離 (Levenshtein Distance) 找出最接近的結果。

## 功能

*   **Levenshtein 距離計算**: 計算兩個字串之間轉換所需的最少編輯操作次數（插入、刪除、替換）。
*   **閾值過濾**: 只顯示編輯距離在設定範圍內的結果。
*   **排序結果**: 根據匹配程度（距離越小越接近）對結果進行排序。

## 使用方式

1.  在 "Items" 文本框中定義要搜尋的資料列表（每行一項）。
2.  在 "Search Query" 輸入框中輸入搜尋關鍵字。
3.  調整 "Max Distance" 以控制模糊匹配的寬鬆程度。
4.  點擊 "Fuzzy Search" 按鈕。
