How it works

  • Converts texts to TF-IDF (Term Frequency-Inverse Document Frequency) vectors.
  • Uses K-Means clustering algorithm to group similar documents.
  • Extracts top keywords for each cluster as cluster labels.
  • Iteratively refines clusters until convergence or max iterations.
