# HTML 實體轉換 (HTML Entity Conversion)

本示例展示了如何使用 Web Workers 進行 HTML 實體的編碼與解碼操作。

## 功能特點

1.  **編碼 (Encode)**:
    - 將 HTML 特殊字符（如 `<` `>` `&` `"` `'` 等）轉換為對應的 HTML 實體（如 `&lt;` `&gt;` `&amp;`）。
    - 防止 XSS 攻擊的基礎步驟。

2.  **解碼 (Decode)**:
    - 將 HTML 實體（包括命名實體與數字實體 `&#x...;`）還原為原始字符。
    - 支援基礎實體與 Unicode 數字實體解析。

3.  **Web Worker**:
    - 所有文本處理邏輯均在 Worker 線程中執行。
    - 確保在處理超長文本時，主線程 UI 不會凍結。

## 技術實現

- **正則表達式**: 使用高效的正則替換進行全域匹配。
- **映射表**: 維護常用實體的映射關係。
- **String.fromCharCode**: 處理數字實體轉換。
