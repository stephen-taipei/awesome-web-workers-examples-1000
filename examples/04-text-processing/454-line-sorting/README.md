# 排序處理 (Line Sorting)

本範例展示如何使用 Web Worker 對文本進行多種方式的排序。

## 功能簡介

- **排序模式**：
    -   **字母順序 (A-Z / Z-A)**：標準的字典序排序。
    -   **長度排序 (短->長 / 長->短)**：按字串長度排序，長度相同時按字典序。
    -   **數值排序**：嘗試將行解析為數字進行比較（例如 "10" > "2"），非數字行排在後面。
    -   **隨機打亂 (Shuffle)**：隨機重排所有行。

## 技術重點

1.  **Array.sort()**：
    -   自定義比較函數（Comparator）以實現不同的排序邏輯。
    -   使用 `localeCompare` 進行正確的字符串比較。

2.  **Web Worker**：
    -   對於包含數萬行的文本，排序（尤其是複雜比較邏輯）可能耗時。Worker 確保了排序過程中 UI 不會卡頓。

## 使用方式

1.  在輸入框中輸入多行文本。
2.  選擇排序方式。
3.  點擊「開始排序」按鈕。
