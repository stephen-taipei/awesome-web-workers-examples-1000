# Camel Case Conversion Example

## Overview
This example demonstrates text conversion to `camelCase` or `PascalCase` using a Web Worker. It processes the input line by line.

## Features
- **camelCase**: firstWordLowercase.
- **PascalCase**: FirstWordUppercase.
- **Delimiters**: Handles spaces, underscores, hyphens, and dots as separators.

## How to Use
1.  Select the desired Output Format.
2.  Enter the text (multiple lines supported).
3.  Click "Convert".
