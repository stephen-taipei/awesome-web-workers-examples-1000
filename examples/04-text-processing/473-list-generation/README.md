# List Generation (Web Worker Example #473)

Converts lines of text into formatted lists (Bulleted, Numbered, HTML, etc.).

## How it works

1.  **Parsing**: Splits input text by newlines and trims whitespace.
2.  **Sorting**: Optionally sorts items (A-Z, Z-A, Length).
3.  **Formatting**: Iterates through items and adds the requested prefix or HTML tags.

## Usage

1.  Enter items (one per line).
2.  Choose list style (e.g., Markdown Bullet `*`, Numbered `1.`, HTML `<ul>`).
3.  Choose sorting preference.
4.  Click "Generate List".
