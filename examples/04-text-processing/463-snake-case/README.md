# Snake Case Conversion Example

## Overview
This example demonstrates text conversion to `snake_case`, `kebab-case`, `SCREAMING_SNAKE_CASE`, or `dot.case` using a Web Worker.

## Features
- **Smart Splitting**: Handles camelCase splitting (e.g., `helloWorld` -> `hello_world`) and existing delimiters.
- **Multiple Formats**: Supports underscore, hyphen, and dot separators.

## How to Use
1.  Select the desired Output Format.
2.  Enter the text.
3.  Click "Convert".
