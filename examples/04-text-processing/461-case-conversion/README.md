# Case Conversion Example

## Overview
This example demonstrates text case conversion using a Web Worker.

## Features
- **Upper/Lower Case**: Standard transformations.
- **Title Case**: Capitalize the first letter of each word.
- **Sentence Case**: Capitalize the start of each sentence.
- **Alternating Case**: aLtErNaTiNg style.
- **Inverse Case**: Swaps the case of each character.

## How to Use
1.  Select the desired Case Mode.
2.  Enter the text.
3.  Click "Convert Case".
