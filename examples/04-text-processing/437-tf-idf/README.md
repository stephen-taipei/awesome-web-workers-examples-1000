# TF-IDF (Term Frequency-Inverse Document Frequency)

Web Worker Example #437

## 簡介

本範例展示如何計算文檔的 TF-IDF 值。TF-IDF 是一種用於資訊檢索與文字探勘的常用加權技術，用於評估一個字詞對於一個檔案集或一個語料庫中的其中一份檔案的重要程度。

## 功能

*   **TF 計算**: 計算詞頻 (Term Frequency)。
*   **IDF 計算**: 計算逆向檔案頻率 (Inverse Document Frequency)。
*   **關鍵詞提取**: 根據 TF-IDF 分數提取每份文檔最重要的關鍵詞。

## 使用方式

1.  在 "Documents" 區域輸入多個文檔（每行一個）。
2.  點擊 "Calculate TF-IDF" 按鈕。
3.  查看每份文檔的前 5 個關鍵詞及其權重。
