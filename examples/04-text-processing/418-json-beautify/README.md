# JSON 美化 (JSON Beautify)

本示例展示了如何使用 Web Workers 對 JSON 數據進行格式化與美化。

## 功能特點

1.  **JSON 格式化**:
    - 將壓縮或混亂的 JSON 字串解析並重新格式化。
    - 支援自定義縮排（2 空格、4 空格、Tab）。

2.  **錯誤處理**:
    - 若輸入的 JSON 格式無效，Worker 會捕獲解析錯誤並回傳給主線程顯示。

3.  **效能**:
    - 對於大型 JSON 文件（如 MB 級別），解析和字串化操作可能會阻塞主線程。使用 Web Worker 可以保持 UI 響應。
