# Sentence Segmentation

此示例展示了如何使用 Web Worker 進行句子分割。

## 技術難點與解決方案

句子分割看似簡單（只需按句號切分），但實際上有許多邊緣情況：
1.  **縮寫詞**：如 "Mr.", "Dr.", "U.S.A." 中的點號不應被視為句尾。
2.  **省略號**：如 "..." 不應切分。
3.  **引號與括號**：標點符號可能包含在引號內。
4.  **多語言支援**：中文的句號 `。`、問號 `？` 和驚嘆號 `！` 不需要空格即可斷句，而英文通常需要後接空格。

## 演算法邏輯

本示例使用正則表達式進行分割：
1.  **保護機制**：暫時替換常見的縮寫點號（如 "Mr." -> "Mr\0"）和省略號，防止誤切。
2.  **插入分隔符**：
    *   對於英文標點 `[.!?]`，僅當其後跟隨空白字符或位於行尾時，才插入分隔符。
    *   對於中文標點 `[。！？]`，直接插入分隔符。
3.  **分割與還原**：按分隔符切分文本，去除空白，並還原之前保護的特殊符號。

## 適用場景

*   機器翻譯預處理
*   文本朗讀 (TTS) 斷句
*   逐句情感分析
