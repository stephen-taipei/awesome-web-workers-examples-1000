# 過濾處理 (Filtering)

本範例展示如何使用 Web Worker 對多行文本進行高效過濾。

## 功能簡介

- **過濾模式**：
    -   **包含關鍵字 (Include)**：保留包含指定關鍵字的行（不區分大小寫）。
    -   **排除關鍵字 (Exclude)**：移除包含指定關鍵字的行。
    -   **正則表達式匹配 (Regex Match)**：保留匹配正則表達式的行。
    -   **正則表達式排除 (Regex Exclude)**：移除匹配正則表達式的行。

## 技術重點

1.  **正則表達式**：
    -   支持用戶輸入任意有效的 JavaScript 正則表達式。
    -   在 Worker 中動態創建 `RegExp` 對象進行匹配。

2.  **Web Worker**：
    -   將遍歷和匹配操作移至後台執行緒，即使處理數萬行文本也不會阻塞 UI。

## 使用方式

1.  在輸入框中輸入多行文本。
2.  選擇過濾模式。
3.  輸入過濾條件（關鍵字或正則表達式）。
4.  點擊「開始過濾」按鈕。
