# Autocomplete (自動完成)

Web Worker Example #435

## 簡介

本範例展示如何使用 Web Worker 和 Trie (前綴樹) 數據結構來實現高效的自動完成功能。Trie 結構非常適合用於前綴匹配搜尋，能夠在大量詞彙中快速找到以特定字元開頭的單詞。

## 功能

*   **Trie 索引構建**: 在 Worker 中構建字典的前綴樹索引。
*   **即時搜尋**: 隨著使用者輸入，即時搜尋並返回建議詞。
*   **效能優化**: 限制返回結果數量，確保搜尋速度極快。

## 使用方式

1.  在 "Dictionary" 區域輸入或貼上單詞列表（支援大量單詞）。
2.  點擊 "Build Trie Index" 按鈕以建立索引。
3.  在 "Type to search" 輸入框中開始輸入，下方會顯示自動完成建議。
