How it works

  • Scans text using a sliding window to find word pairs that appear together.
  • Window size determines how many words apart can be considered co-occurring.
  • Filters out low-frequency words to focus on meaningful patterns.
  • Matrix shows normalized co-occurrence counts (color intensity = frequency).
