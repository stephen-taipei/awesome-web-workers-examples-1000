How it works

  • The main thread sends raw amounts to a Web Worker.
  • The Worker uses `Intl.NumberFormat` with `style: 'currency'` for correct financial formatting.
  • Handles currency codes, symbols, and locale-specific placement (prefix vs suffix).
  • Supports crypto currencies (like BTC) if browser supports it.
