# Citation Formatting (Web Worker Example #478)

Formats bibliographic data into standard citation styles (APA, MLA, Chicago, IEEE).

## How it works

1.  **Parsing**: Reads the input JSON array of citation objects.
2.  **Formatting**: Applies style-specific templates to rearrange authors, titles, dates, and publication info.
3.  **Output**: Generates formatted strings (with Markdown italics `*`).

## Usage

1.  Input citation data in JSON format (array of objects).
2.  Select the desired citation style.
3.  Click "Format Citations".
