How it works

  • The main thread sends a list of integers to a Web Worker.
  • The Worker uses a recursive algorithm to convert numbers (up to quadrillions) into English words.
  • Example: `123` -> "one hundred twenty-three".
  • Handles large datasets efficiently without blocking the UI.
