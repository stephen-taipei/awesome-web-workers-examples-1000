# 中文分詞 (Chinese Word Segmentation)

本範例展示如何使用 Web Worker 實現基礎的中文分詞功能。由於中文不像英文有空格作為自然分隔符，因此需要特定的算法來切分詞彙。

## 功能簡介

- **最大匹配算法**：實現了最基礎的兩種匹配算法：
    -   **正向最大匹配 (FMM)**：從左向右掃描，優先匹配最長的詞。
    -   **逆向最大匹配 (BMM)**：從右向左掃描，優先匹配最長的詞。
- **內建詞典**：Worker 中包含一個極簡的示範詞典。在實際生產環境中，這通常會是一個龐大的 Trie 樹結構或雙數組 Trie。

## 技術重點

1.  **字典查找**：
    -   使用 JavaScript `Set` 進行 $O(1)$ 的快速查找。
    -   實際應用中可擴展為 Trie 樹以節省內存並支持前綴查詢。

2.  **Web Worker**：
    -   分詞過程涉及大量的字串操作和循環，對於長文本（如小說、文章），移至 Worker 執行可確保 UI 流暢。

## 使用方式

1.  在文本框中輸入中文句子。
2.  選擇分詞算法（FMM 或 BMM）。
3.  點擊「開始分詞」按鈕。
4.  結果將以區塊形式顯示，每個區塊代表一個分出的詞。
