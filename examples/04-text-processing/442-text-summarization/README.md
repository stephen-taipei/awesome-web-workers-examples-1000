# Text Summarization

此示例展示了如何使用 Web Worker 實現提取式文本摘要（Extractive Summarization）。

## 技術原理

本示例使用類似 TextRank 的簡化算法來評估句子的重要性：

1.  **句子分割**：將文章分割成獨立的句子。
2.  **預處理**：將句子分詞並移除停用詞（Stop Words），提取關鍵內容詞。
3.  **相似度計算**：計算每對句子之間的相似度。這裡使用基於共同詞彙數量的指標（類似 Jaccard 係數）。
    *   如果兩個句子包含許多相同的關鍵詞，它們的相似度就高。
4.  **評分**：一個句子的得分是它與所有其他句子相似度的總和。
    *   這基於一個假設：與文章中其他部分最相關的句子，最能代表文章的核心內容。
5.  **排序與提取**：選取得分最高的 N 個句子，並按其在原文中的出現順序重新排列，生成連貫的摘要。

## 適用場景

*   長篇文章快速預覽
*   新聞聚合
*   自動生成報告摘要

## 注意事項

*   這是一種「提取式」摘要，不會重寫句子，只會挑選原文中的句子。
*   對於需要理解深層語意或重組語言的任務，通常需要更複雜的 NLP 模型（如 BERT, GPT 等），這在瀏覽器端運行成本較高。
