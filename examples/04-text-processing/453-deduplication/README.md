# 去重處理 (Deduplication)

本範例展示如何使用 Web Worker 對文本進行行級去重 (Remove Duplicate Lines)。

## 功能簡介

- **去重**：移除文本中重複出現的行，僅保留第一次出現的行。
- **選項**：
    -   **忽略大小寫**：視 "Apple" 和 "apple" 為相同。
    -   **去除空白**：在比較前去除每行首尾的空白字符。

## 技術重點

1.  **Set 資料結構**：
    -   利用 JavaScript 的 `Set` 對象存儲已出現過的行（或其 Hash/Key），實現 $O(1)$ 複雜度的查找。
    -   整體時間複雜度為 $O(N)$，其中 N 為文本行數。

2.  **Web Worker**：
    -   處理數十萬行的大文本時，主執行緒可能會凍結。Worker 確保了大型數據集處理時的響應性。

## 使用方式

1.  在輸入框中粘貼或輸入多行文本。
2.  勾選需要的選項（忽略大小寫、去除空白）。
3.  點擊「去除重複」按鈕。
4.  查看結果及行數統計。
