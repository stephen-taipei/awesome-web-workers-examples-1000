# SQL 格式化 (SQL Formatter)

本示例展示了如何使用 Web Workers 對 SQL 查詢語句進行基礎格式化。

## 功能特點

1.  **關鍵字大寫**:
    - 自動將標準 SQL 關鍵字（如 `select`, `from`, `where`）轉換為大寫。

2.  **排版優化**:
    - 在主要子句（`SELECT`, `FROM`, `WHERE` 等）前添加換行。
    - 對 `JOIN` 和 `AND/OR` 條件進行適當的縮排處理。

## 注意事項

- 本格式化器基於關鍵字替換，不具備完整的 SQL 語法解析能力。
- 可能會錯誤地大寫字符串常量中的單詞（如果它們恰好是關鍵字）。
- 適用於標準 SQL 的快速美化。
