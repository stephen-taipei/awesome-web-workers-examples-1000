How it works

  • The main thread sends raw date strings to a Web Worker.
  • The Worker parses each date (handling standard formats and timestamps).
  • Dates are formatted using `Intl.DateTimeFormat` or custom logic (for ISO/Relative).
  • The processed list is returned efficiently.
  • Supports processing 100,000+ dates rapidly.
