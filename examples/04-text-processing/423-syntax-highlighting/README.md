# 程式碼高亮 (Syntax Highlighting)

本示例展示了如何使用 Web Workers 對程式碼進行簡單的語法高亮處理。

## 功能特點

1.  **多語言支援**:
    - 支援 JavaScript, Python, CSS, HTML 的基礎語法高亮。
    - 識別關鍵字、字串、註釋、數字和內建對象。

2.  **HTML 生成**:
    - 將源代碼轉換為帶有 CSS 類別的 HTML 片段。
    - 防止 XSS (自動轉義 HTML 特殊字符)。

3.  **非阻塞**:
    - 即使處理長代碼文件，也能保持頁面響應。

## 注意事項

- 本高亮器基於正則表達式詞法分析 (Lexical Analysis)，並非完整的語法解析器。
- 可能會出現邊緣情況錯誤（例如正則表達式在字符串內部的匹配問題，雖然算法嘗試了解決）。
- 對於生產環境，建議使用 PrismJS 或 Highlight.js 等成熟庫。
