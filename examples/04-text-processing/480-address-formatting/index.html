How it works

  • The main thread sends the raw text to a Web Worker.
  • The Worker parses each line using Regex to identify street, city, state, zip code (US format focus for this demo).
  • Addresses are standardized (e.g., "St." -> "Street", capitalizing properly).
  • The formatted result is sent back to the main thread in chunks or as a whole.
