# 程式碼格式化 (Code Formatter)

本示例展示了如何使用 Web Workers 對 C-style 語言（如 Java, JS, C#, CSS）的程式碼進行基礎格式化。

## 功能特點

1.  **自動縮排**:
    - 基於花括號 `{}` 的層級進行自動縮排。
    - 支援自定義縮排單位（空格或 Tab）。

2.  **語句分行**:
    - 自動在分號 `;` 和花括號後換行。
    - 保持字串和註釋內容的完整性。

## 注意事項

- 本格式化器是基於正則和符號的簡易版本，不具備完整的抽象語法樹 (AST) 解析能力。
- 對於某些複雜語法（如 Switch case, 箭頭函數隱式返回, 模板字串中的特殊字符）可能格式化不完美。
- 主要用途是快速整理壓縮過的或混亂的代碼塊。
