How it works

  • Uses the Intl.Collator API for locale-aware string comparison.
  • Properly handles accented characters, non-Latin scripts, and numeric strings.
  • Supports multiple sensitivity levels for different comparison needs.
  • Example: In Swedish, "o" and "o" are different letters; in German, "ss" equals "ss".
