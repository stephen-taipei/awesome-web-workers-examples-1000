# Word Count

此示例展示了如何使用 Web Worker 進行詳細的文本統計。

## 統計指標

*   **字元數 (含空格)**：文本的總長度。
*   **字元數 (不含空格)**：移除所有空白字元後的長度。
*   **單詞數 (Words)**：
    *   英文單詞：由字母組成的連續序列。
    *   CJK 字元：每個中日韓漢字、假名、諺文單獨視為一個單詞。
    *   總數 = 英文單詞數 + CJK 字元數。
*   **行數 (Lines)**：由換行符分隔的行數。
*   **段落數 (Paragraphs)**：由空行分隔的文本塊數量。
*   **中日韓字元**：專門統計 CJK 範圍內的字元數量。
*   **平均詞長**：英文單詞的平均字母數。

## 適用場景

*   編輯器狀態列
*   SEO 內容分析
*   閱讀時間估算
