# Spelling Suggestion (拼寫建議)

Web Worker Example #434

## 簡介

本範例展示如何實現一個基於字典的拼寫建議系統。當使用者輸入一個單詞時，系統會計算該單詞與字典中所有單詞的編輯距離，並推薦最相似的正確拼寫。

## 功能

*   **字典庫**: 內建常用單詞字典，並支援使用者自訂。
*   **距離計算**: 使用 Levenshtein 距離算法評估單詞相似度。
*   **智慧推薦**: 根據相似度排序並過濾，只顯示最相關的建議。

## 使用方式

1.  在 "Dictionary" 區域可以編輯或替換預設的單詞庫。
2.  在 "Misspelled Word" 輸入框中輸入一個可能拼錯的單詞。
3.  點擊 "Get Suggestions" 按鈕。
4.  點擊推薦結果可以直接填入輸入框。
