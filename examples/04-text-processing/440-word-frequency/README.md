# Word Frequency

此示例展示了如何使用 Web Worker 進行文本詞頻統計。

## 技術重點

*   **正則表達式分詞**：使用正則表達式 `[\w\u4e00-\u9fa5]+` 來提取文本中的單詞（支援英文與基本中文）。
*   **Map 統計**：使用 JavaScript 的 `Map` 資料結構來高效存儲和更新詞頻。
*   **排序**：將統計結果轉換為陣列並按頻率降序排列，提取前 100 個高頻詞。
*   **背景處理**：大量的文本處理（分詞、遍歷、排序）在 Worker 中執行，避免阻塞主介面。

## 適用場景

*   文章關鍵詞分析
*   標籤雲生成
*   文本特徵提取

## 注意事項

此範例使用簡易的分詞規則。對於中文等無空格分隔的語言，精確的分詞通常需要專門的斷詞庫（如 Jieba），這裡僅將連續的漢字或英數字視為詞彙單元。
