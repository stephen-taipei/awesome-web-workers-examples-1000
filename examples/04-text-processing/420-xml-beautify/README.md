# XML 美化 (XML Beautify)

本示例展示了如何使用 Web Workers 對 XML 數據進行格式化與美化。

## 功能特點

1.  **XML 格式化**:
    - 將緊湊的 XML 字串重新排版，增加縮排與換行。
    - 支援自定義縮排（2 空格、4 空格、Tab）。

2.  **簡易解析**:
    - 使用正則表達式進行標籤識別與層級追蹤。
    - 適用於大多數標準格式的 XML 數據。

## 注意事項

- 本示例使用的是基於正則的簡易格式化器，對於包含複雜 CDATA、混合內容或特殊格式的 XML，可能會有些許排版誤差。
- 主要用於數據型 XML 的可讀性優化。
