# Keyword Extraction

此示例展示了如何使用 Web Worker 實現關鍵詞提取功能，採用了 RAKE (Rapid Automatic Keyword Extraction) 演算法的簡化版本。

## 技術原理 (RAKE)

RAKE 是一種基於圖論和統計的無監督關鍵詞提取算法，特別適合處理單個文檔。

主要步驟：
1.  **句子分割**：將文本分割成句子。
2.  **候選詞生成**：利用「停用詞（Stop Words）」作為分隔符，將句子切分為多個短語（Candidate Keywords）。例如 "deep learning algorithms" 是一個短語，如果前後有停用詞。
3.  **詞彙評分**：
    *   計算每個詞的 **頻率 (Frequency)**：在所有短語中出現的次數。
    *   計算每個詞的 **度 (Degree)**：該詞在短語中與其他詞共現的次數總和（加上自身頻率）。
    *   **得分 = Degree / Frequency**。傾向於選出長短語中常出現的詞。
4.  **短語評分**：將短語中所有單詞的得分相加。
5.  **排序**：按得分降序排列。

## 適用場景

*   SEO 關鍵詞分析
*   文章標籤自動生成
*   文檔摘要輔助

## 注意事項

*   本示例內建了一份簡單的英文停用詞表。
*   對於中文提取，需要更複雜的分詞與中文停用詞表，本示例目前主要針對英文內容優化。
