# Paragraph Segmentation

此示例展示了如何使用 Web Worker 進行段落分割。

## 技術原理

在純文本中，段落通常由一個或多個空行（Double Newline）分隔。
本示例的 Worker 腳本執行以下步驟：

1.  **換行符標準化**：將所有類型的換行符（`\r\n`, `\r`）統一轉換為 `\n`。
2.  **正則分割**：使用正則表達式 `/\n\s*\n/` 進行分割。
    *   這不僅能識別連續的兩個換行符，還能識別中間包含空白字符（如空格、Tab）的空行。
3.  **過濾**：移除分割後可能產生的空字符串或僅包含空白的段落。

## 適用場景

*   文本排版與渲染
*   文章結構分析
*   Markdown 解析器預處理
