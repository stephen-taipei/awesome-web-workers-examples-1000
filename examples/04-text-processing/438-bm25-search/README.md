# BM25 Search (BM25 搜尋)

Web Worker Example #438

## 簡介

本範例展示如何使用 Web Worker 實現 BM25 搜尋算法。BM25 (Best Matching 25) 是一種概率資訊檢索模型，被廣泛認為是最先進的排名函數之一，通常優於標準的 TF-IDF。

## 功能

*   **BM25 索引**: 計算文檔長度、平均文檔長度和 IDF 值。
*   **相關性評分**: 根據查詢詞頻、文檔長度和全局頻率計算相關性分數。
*   **結果排序**: 按分數從高到低返回最相關的文檔。

## 使用方式

1.  在 "Documents" 區域輸入 JSON 字符串數組格式的文檔集。
2.  點擊 "Index Documents" 建立索引。
3.  在搜尋框輸入查詢語句，點擊 "Search"。
