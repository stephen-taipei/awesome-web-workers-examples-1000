# Pinyin Conversion Example

## Overview
This example demonstrates text conversion from Chinese characters to Pinyin using a Web Worker.

## Features
- **Tone Marks**: Optionally display tone marks (e.g., h«éo vs hao).
- **Web Worker**: Performs dictionary lookup and formatting off the main thread.

## Note
This example includes a **small demonstration dictionary**. It does not cover the full set of Chinese characters.

## How to Use
1.  Enter the Chinese text to convert.
2.  Choose whether to include tone marks.
3.  Click "Convert to Pinyin".
